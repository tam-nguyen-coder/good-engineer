# Amazon SageMaker

- Unified analytics and AI

- Analytics and AI development

- Optimized for using a single development environment, Amazon SageMaker Unified Studio, to access data, analytics, and AI capabilities

Amazon SageMaker is a fully managed service that provides every developer and data scientist with the ability to build, train, and deploy machine learning models quickly.


# Keywords

- **Build, Train, Deploy** (End-to-end Machine Learning lifecycle management)
- **SageMaker Studio** (Single web-based visual interface/IDE for ML development)
- **SageMaker Autopilot** (AutoML capability to build, train, and tune best ML models automatically)
- **SageMaker Ground Truth** (High-quality data labeling tool using ML and human labelers)
- **SageMaker Neo** (Compiles and optimizes ML models to run on edge devices or EC2 instances with less resources)
- **SageMaker Model Monitor** (Monitors quality of deployed ML models, detects data drift)
- **SageMaker Clarify** (Detects bias in ML models and explains predictions)
