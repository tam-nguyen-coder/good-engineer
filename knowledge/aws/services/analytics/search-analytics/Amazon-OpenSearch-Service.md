# Amazon OpenSearch Service

- Search Analytics

- Managed OpenSearch clusters

Optimized for log analytics, real-time application monitoring, and clickstream analysis.

# Keywords

- **Log Analytics** (Primary use case for analyzing system, application, and VPC Flow Logs)
- **Elasticsearch / Kibana** (OpenSearch is successor to Amazon Elasticsearch Service; Kibana is now OpenSearch Dashboards)
- **Real-time Application Monitoring** (Interactive search and analysis of operational data)
- **OpenSearch Dashboards** (Built-in visualization tool for search results and metrics)
- **Cognito Authentication** (Integrates with Amazon Cognito to secure access to OpenSearch Dashboards)
- **Firehose Destination** (Native integration as a direct destination for Amazon Data Firehose streams)
- **VPC Deployment** (Can be deployed inside a VPC for private network security)
