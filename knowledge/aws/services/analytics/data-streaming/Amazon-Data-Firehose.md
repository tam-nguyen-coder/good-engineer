# Amazon Data Firehose

- Data Streaming

- Real-time streaming data delivery

Optimized for delivering real-time streaming data to destinations such as Amazon S3, Amazon Redshift, OpenSearch Service, Splunk, Apache Iceberg Tables, and any custom HTTP endpoint or HTTP endpoints owned by supported third-party service providers.

# Keywords

- **Delivery Stream**
- **S3 / Redshift / OpenSearch / Splunk / Apache Iceberg / HTTP Endpoints**
- **Data Transformation** (via Lambda)
- **Buffering** (Buffer size / Buffer interval)
- **Fully Managed / Serverless** (auto-scaling)
- **Near Real-time**
