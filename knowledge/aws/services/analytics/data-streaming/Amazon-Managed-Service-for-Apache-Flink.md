# Amazon Managed Service for Apache Flink

- Data Streaming

- Building Apache Flink applications.

Optimized for using Java, Scala, Python, or SQL to process and analyze streaming data.

# Keywords

- **Apache Flink** (Open-source framework and engine for stateful computations over data streams)
- **Kinesis Data Analytics** (Former name of the service)
- **Time Windows (Tumbling, Hopping, Sliding, Session)** (Aggregating stream data over time-based windows)
- **SQL / Java / Scala / Python** (Supported languages to write stream processing applications)
- **Serverless / KPUs (Kinesis Processing Units)** (Automatically scales capacity based on processing needs)
- **Source & Sink Integration** (Reads from Kinesis Streams/MSK; writes to Firehose, KDS, S3, etc.)
- **Real-time Analytics** (Low-latency stream processing and analysis)
