# Câu hỏi #48

In the AWS Shared responsibility Model, which of the following are the responsibility of the customer? (Choose TWO)

## Options

**A.** Disk disposal.
**B.** Controlling physical access to compute resources.
**C.** Patching the Network infrastructure.
**D.** Setting password complexity rules.
**E.** Configuring network access rules.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Đề hỏi về trách nhiệm của **khách hàng** trong `AWS Shared Responsibility Model`.
- **Existing Resources:** Không có tài nguyên cụ thể.
- **Current Issue/Goal:** Xác định 2 hành động thuộc trách nhiệm khách hàng.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `Disk disposal` | AWS chịu (lớp vật lý); nếu dùng EBS bạn chịu wiping data trước khi terminate |
| `Physical access` | AWS chịu hoàn toàn |
| `Patching Network infrastructure` | AWS chịu (router, switch, ...) |
| `Setting password complexity rules` | Khách hàng chịu — liên quan IAM |
| `Configuring network access rules` | Khách hàng chịu — Security Group, NACL |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Multi-answer (Choose TWO) — Customer responsibility
- **Constraints:** Phải là hành động trong tầm kiểm soát của khách hàng

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: D và E**

**Giải thích:**
- **D — Setting password complexity rules:** Khách hàng chịu trách nhiệm cấu hình `IAM password policy` (độ dài tối thiểu, ký tự đặc biệt, thời hạn hết hạn, ...). Đây là một phần của "Security IN the cloud".
- **E — Configuring network access rules:** Khách hàng chịu trách nhiệm cấu hình `Security Group`, `Network ACL`, `Route Table` để kiểm soát traffic vào/ra EC2 và subnet. Đây là lớp bảo mật khách hàng tự quản lý.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- `Disk disposal` (tiêu hủy vật lý disk lỗi): AWS chịu trách nhiệm tiêu hủy disk vật lý theo chuẩn NIST 800-88. Lưu ý: nếu bạn dùng `EBS`, việc **xóa data nhạy cảm trước khi terminate** vẫn thuộc trách nhiệm của bạn.

**❌ Đáp án B:**
- `Controlling physical access to compute resources` hoàn toàn thuộc AWS — bao gồm khoá phòng server, xác thực đa lớp, giám sát CCTV tại data center.

**❌ Đáp án C:**
- `Patching the Network infrastructure` thuộc AWS — bao gồm router, switch, firewall vật lý, hypervisor. Khách hàng không truy cập được lớp này.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *"Khách hàng chịu: `password policy` + `network rules` (SG/NACL). AWS chịu: physical access, disk disposal, network patching."*
