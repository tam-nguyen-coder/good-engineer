# Câu hỏi #33

Your company has a data store application that requires access to a NoSQL database. Which AWS database offering would meet this requirement?

## Options

**A.** Amazon Aurora.
**B.** Amazon DynamoDB.
**C.** Amazon Elastic Block Store.
**D.** Amazon Redshift.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Công ty có một **data store application** cần truy cập **NoSQL database**.
- **Existing Resources:** Application cần chọn database backend.
- **Current Issue/Goal:** Chọn dịch vụ database NoSQL của AWS.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `NoSQL database` | Cơ sở dữ liệu không quan hệ (key-value, document, graph, ...) |
| `Amazon DynamoDB` | NoSQL key-value & document database, fully managed |
| `Amazon Aurora` | Relational (SQL) database, tương thích MySQL/PostgreSQL |
| `Amazon EBS` | Block storage cho EC2, không phải database |
| `Amazon Redshift` | Data warehouse (columnar SQL), dùng cho analytics |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Chọn database service phù hợp
- **Constraints:** Phải là NoSQL

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: B**

**Giải thích:**
- `Amazon DynamoDB` là dịch vụ `NoSQL database` của AWS, hỗ trợ mô hình dữ liệu key-value và document, với hiệu năng ổn định ở mọi quy mô (single-digit millisecond).
- Fully managed: AWS tự động scale, replicate across 3 AZ, backup, và cung cấp tính năng `DynamoDB Streams`, `Global Tables`, `DAX` (in-memory accelerator).
- Phù hợp cho game, IoT, mobile, web app cần đọc/ghi nhanh với schema linh hoạt.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- `Amazon Aurora` là relational database (RDBMS) tương thích MySQL/PostgreSQL, **không phải NoSQL**.

**❌ Đáp án C:**
- `Amazon EBS` (Elastic Block Store) là block storage gắn với EC2, không phải database.

**❌ Đáp án D:**
- `Amazon Redshift` là data warehouse dựa trên PostgreSQL, dùng cho phân tích dữ liệu lớn (OLAP), không phải NoSQL cho transactional workload.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *"NoSQL trong AWS = `DynamoDB` (key-value + document). `Aurora` = relational, `Redshift` = data warehouse, `EBS` = block storage. Phân biệt rõ!"*
