# Câu hỏi #594

Your customer wants to grant restricted proxy rights to their HR Department to act on behalf of employees belonging to their legal entity. How can you perform this requirement? There are 2 correct answers to this question.

## Options

**A.** Use the Manage Permission Roles tool to grant proxy rights to the HR Department.
**B.** Fill the PROXY column of the Basic Import template to match the requirements and import the file using the Import Employee Data tool.
**C.** Use the Proxy Management tool to configure the requested proxy assignments.
**D.** Fill the Basic Import template with the requested proxy configuration and import the file using the Import and Export Data tool.

**Giải thích:**

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- Tại sao sai: `IAM` (Identity and Access Management - quản lý quyền truy cập) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu bảo mật chính là quản lý user, group, role, policy cho AWS resources này.

**❌ Đáp án D:**
- Tại sao sai: Đáp án này không trực tiếp giải quyết yêu cầu cụ thể của đề, hoặc dịch vụ/tính năng này có giới hạn so với đáp án đúng.
- Khi nào đúng: Đáp án này sẽ đúng trong bối cảnh khác, nơi ưu tiên khác với yêu cầu của câu hỏi này.
