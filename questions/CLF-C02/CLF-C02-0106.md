# Câu hỏi #106

A company is developing a new application using a microservices framework. The new application is having performance and latency issues. Which AWS Service should be used to troubleshoot these issues?

## Options

**A.** AWS CodePipeline.
**B.** AWS X-Ray.
**C.** Amazon Inspector.
**D.** AWS CloudTrail.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Tình huống được mô tả: _A company is developing a new application using a microservices framework. The new application is having performance and..._
- **Existing Resources:** Không có tài nguyên cụ thể — câu hỏi thuộc dạng lý thuyết/khái niệm.
- **Current Issue/Goal:** Lựa chọn giải pháp có hiệu năng cao nhất, độ trễ thấp nhất.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| ``CloudTrail`` | dịch vụ audit API call |
| ``Inspector`` | dịch vụ quét lỗ hổng bảo mật tự động |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Best performance / Lowest latency
- **Constraints:** Hiệu năng và độ trễ thấp

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: B**

**Giải thích:**
Đáp án **B** mô tả đúng khái niệm/dịch vụ mà đề đang hỏi tới.
Đây là đáp án phù hợp nhất với best practice của AWS trong tình huống này.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- Tại sao sai: Đáp án này không trực tiếp giải quyết yêu cầu cụ thể của đề, hoặc dịch vụ/tính năng này có giới hạn so với đáp án đúng.
- Khi nào đúng: Đáp án này sẽ đúng trong bối cảnh khác, nơi ưu tiên khác với yêu cầu của câu hỏi này.

**❌ Đáp án C:**
- Tại sao sai: `Inspector` (dịch vụ quét lỗ hổng bảo mật tự động) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu đánh giá security posture của EC2, container, Lambda được ưu tiên hơn hiệu năng.

**❌ Đáp án D:**
- Tại sao sai: `CloudTrail` (dịch vụ audit API call) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu ghi lại mọi API call vào tài khoản AWS, hỗ trợ kiểm tra bảo mật được ưu tiên hơn hiệu năng.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **B** = `CloudTrail`. Hiệu năng = edge location + cache + chọn instance/region gần user.*
