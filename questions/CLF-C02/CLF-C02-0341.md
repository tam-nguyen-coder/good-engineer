# Câu hỏi #341

Which AWS service allows you to build a data warehouse in the cloud?

## Options

**A.** AWS Shield.
**B.** Amazon Redshift.
**C.** Amazon RDS.
**D.** Amazon Comprehend.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Đề hỏi về khái niệm/dịch vụ AWS liên quan tới: _Which AWS service allows you to build a data warehouse in the cloud?_
- **Existing Resources:** Đề nhắc tới: `Redshift` — data warehouse columnar.
- **Current Issue/Goal:** Lựa chọn đáp án đúng nhất theo yêu cầu của đề.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `Big data / Analytics` | Phân tích dữ liệu lớn |
| ``RDS`` | dịch vụ database quan hệ được quản lý |
| ``Shield`` | dịch vụ chống DDoS |
| ``Redshift`` | data warehouse columnar |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Database
- **Constraints:** Tuân theo best practice AWS

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: B**

**Giải thích:**
Đáp án `Redshift` đúng vì data warehouse columnar, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `Redshift` đóng vai trò: phân tích dữ liệu lớn, OLAP, BI.

- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `RDS`.
