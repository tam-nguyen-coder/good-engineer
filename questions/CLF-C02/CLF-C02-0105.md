# Câu hỏi #105

What is the primary storage service used by Amazon RDS database instances?

## Options

**A.** Amazon Glacier.
**B.** Amazon EBS.
**C.** Amazon EFS.
**D.** Amazon S3.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Đề hỏi về khái niệm/dịch vụ AWS liên quan tới: _What is the primary storage service used by Amazon RDS database instances?_
- **Existing Resources:** Đề nhắc tới: `RDS` — dịch vụ database quan hệ được quản lý.
- **Current Issue/Goal:** Xác định đúng khái niệm/dịch vụ/tính năng AWS được hỏi tới.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| ``S3`` | dịch vụ lưu trữ object bền vững (11x 9s) |
| ``RDS`` | dịch vụ database quan hệ được quản lý |
| ``Glacier`` | dịch vụ lưu trữ lạnh giá rẻ |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Storage
- **Constraints:** Tuân theo best practice AWS

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: B**

**Giải thích:**
Đáp án `EBS` đúng vì Elastic Block Store - ổ đĩa block cho EC2, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `EBS` đóng vai trò: lưu trữ persistent cho EC2, snapshot, multi-attach.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- Tại sao sai: `Glacier` (dịch vụ lưu trữ lạnh giá rẻ) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `Glacier`.

**❌ Đáp án C:**
- Tại sao sai: `EFS` (Elastic File System - file system NFS managed) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `EFS`.

**❌ Đáp án D:**
- Tại sao sai: `S3` (dịch vụ lưu trữ object bền vững (11x 9s)) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `S3`.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **B** = `S3` — phù hợp với yêu cầu đề bài.*
