# Câu hỏi #268

Which statement is true in relation to the security of Amazon EC2?

## Options

**A.** You should use instance store volumes to store login data.
**B.** You should regularly patch the operating system and applications on your EC2 instances.
**C.** You should deploy critical components of your application in the Availability Zone that you trust.
**D.** You can track all API calls using Amazon Athena.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Đề hỏi về khái niệm/dịch vụ AWS liên quan tới: _Which statement is true in relation to the security of Amazon EC2?_
- **Existing Resources:** Đề nhắc tới: `EC2` — dịch vụ compute (máy chủ ảo) trên AWS Cloud.
- **Current Issue/Goal:** Lựa chọn giải pháp bảo mật tốt nhất theo best practice của AWS.

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Most secure / Security best practice
- **Constraints:** Bảo mật dữ liệu và quyền truy cập

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: B**

**Giải thích:**
Đáp án `EC2` đúng vì dịch vụ compute (máy chủ ảo) trên AWS Cloud, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `EC2` đóng vai trò: chạy ứng dụng, web server, batch processing.

**❌ Đáp án A:**
- Tại sao sai: Đáp án này không trực tiếp giải quyết yêu cầu cụ thể của đề, hoặc dịch vụ/tính năng này có giới hạn so với đáp án đúng.

**❌ Đáp án C:**
- Tại sao sai: Đáp án này không trực tiếp giải quyết yêu cầu cụ thể của đề, hoặc dịch vụ/tính năng này có giới hạn so với đáp án đúng.
- Khi nào đúng: Đáp án này sẽ đúng trong bối cảnh khác, nơi ưu tiên khác với yêu cầu của câu hỏi này.

**❌ Đáp án D:**
- Tại sao sai: `CloudTrail` (dịch vụ audit API call) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu bảo mật chính là ghi lại mọi API call vào tài khoản AWS, hỗ trợ kiểm tra bảo mật này.
