# Câu hỏi #60

You work as an on-premises MySQL DBA. The work of database configuration, backups, patching, and DR can be time-consuming and repetitive. Your company has decided to migrate to the AWS Cloud. Which of the following can help save time on database maintenance so you can focus on data architecture and performance?

## Options

**A.** Amazon RDS.
**B.** Amazon Redshift.
**C.** Amazon DynamoDB.
**D.** Amazon CloudWatch.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Tình huống được mô tả: _You work as an on-premises MySQL DBA. The work of database configuration, backups, patching, and DR can be time-consumin..._
- **Existing Resources:** Không có tài nguyên cụ thể — câu hỏi thuộc dạng lý thuyết/khái niệm.
- **Current Issue/Goal:** Lựa chọn giải pháp có hiệu năng cao nhất, độ trễ thấp nhất.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `Disaster recovery` | Khôi phục sau thảm họa |
| `Hybrid / On-premises` | Kết nối môi trường on-premises với AWS |
| `Backup / Archive` | Sao lưu, lưu trữ dài hạn |
| ``RDS`` | dịch vụ database quan hệ được quản lý |
| ``DynamoDB`` | dịch vụ NoSQL key-value được quản lý hoàn toàn |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Best performance / Lowest latency
- **Constraints:** Hiệu năng và độ trễ thấp

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: A**

**Giải thích:**
Đáp án `RDS` đúng vì dịch vụ database quan hệ được quản lý, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `RDS` đóng vai trò: chạy SQL database (MySQL, PostgreSQL, Oracle, SQL Server, MariaDB, Aurora).

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án B:**
- Tại sao sai: `Redshift` (data warehouse columnar) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu phân tích dữ liệu lớn, OLAP, BI được ưu tiên hơn hiệu năng.

**❌ Đáp án C:**
- Tại sao sai: `DynamoDB` (dịch vụ NoSQL key-value được quản lý hoàn toàn) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu lưu trữ dữ liệu phi quan hệ với độ trễ thấp ở mọi quy mô được ưu tiên hơn hiệu năng.

**❌ Đáp án D:**
- Tại sao sai: `CloudWatch` (dịch vụ monitoring và observability) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu thu thập metric, log, set alarm, dashboard được ưu tiên hơn hiệu năng.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **A** = `RDS`. Hiệu năng = edge location + cache + chọn instance/region gần user.*
