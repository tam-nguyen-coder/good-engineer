# Câu hỏi #560

How can one AWS account use Reserved Instances from another AWS account?

## Options

**A.** By using Amazon EC2 Dedicated Instances.
**B.** By using AWS Organizations consolidated billing.
**C.** By using the AWS Cost Explorer tool.
**D.** By using AWS Budgets.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Đề hỏi về khái niệm/dịch vụ AWS liên quan tới: _How can one AWS account use Reserved Instances from another AWS account?_
- **Existing Resources:** Không có tài nguyên cụ thể — câu hỏi thuộc dạng lý thuyết/khái niệm.
- **Current Issue/Goal:** Lựa chọn giải pháp giúp tối ưu/tiết kiệm chi phí nhất mà vẫn đáp ứng yêu cầu nghiệp vụ.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| ``EC2`` | dịch vụ compute (máy chủ ảo) trên AWS Cloud |
| ``Organizations`` | quản lý nhiều tài khoản AWS |
| ``Cost Explorer`` | công cụ trực quan hóa chi phí |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Most cost-effective / Cost optimization
- **Constraints:** Ngân sách/chi phí là yếu tố quan trọng

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: B**

**Giải thích:**
Đáp án `Organizations` đúng vì quản lý nhiều tài khoản AWS, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `Organizations` đóng vai trò: gộp tài khoản, áp dụng SCP, tập trung billing.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- Tại sao sai: `EC2` (dịch vụ compute (máy chủ ảo) trên AWS Cloud) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi ưu tiên chạy ứng dụng, web server, batch processing mà không yêu cầu tối ưu chi phí.

**❌ Đáp án C:**
- Tại sao sai: `Cost Explorer` (công cụ trực quan hóa chi phí) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi ưu tiên xem, phân tích chi phí theo service, thời gian mà không yêu cầu tối ưu chi phí.

**❌ Đáp án D:**
- Tại sao sai: `AWS Budgets` (thiết lập ngân sách và cảnh báo) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi ưu tiên alert khi chi phí vượt ngưỡng đã đặt mà không yêu cầu tối ưu chi phí.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **B** = `EC2`. Tiết kiệm = pay-as-you-go, đừng mua hạ tầng thừa.*
