# Câu hỏi #572

Which of the following AWS features enables a user to launch a pre-configured Amazon Elastic Compute Cloud (Amazon EC2) instance?

## Options

**A.** Amazon Elastic Block Store (Amazon EBS).
**B.** Amazon Machine Image.
**C.** Amazon EC2 Systems Manager.
**D.** Amazon AppStream 2.0.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Ngữ cảnh đề bài: _Which of the following AWS features enables a user to launch a pre-configured Amazon Elastic Compute Cloud (Amazon EC2) ..._
- **Existing Resources:** Đề nhắc tới: `EC2` — dịch vụ compute (máy chủ ảo) trên AWS Cloud.
- **Current Issue/Goal:** Hệ thống cần co giãn linh hoạt theo tải, không bị giới hạn bởi cấu hình cố định.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `Real-time` | Xử lý thời gian thực |
| ``EC2`` | dịch vụ compute (máy chủ ảo) trên AWS Cloud |
| ``EBS`` | Elastic Block Store - ổ đĩa block cho EC2 |
| ``Systems Manager`` | quản lý fleet EC2 và on-premises |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Scalability / Elasticity
- **Constraints:** Khả năng mở rộng theo tải

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: B**

**Giải thích:**
Đáp án **B** mô tả đúng khái niệm/dịch vụ mà đề đang hỏi tới.
Đây là đáp án phù hợp nhất với best practice của AWS trong tình huống này.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- Tại sao sai: `EBS` (Elastic Block Store - ổ đĩa block cho EC2) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `EBS`.

**❌ Đáp án C:**
- Tại sao sai: `EC2` (dịch vụ compute (máy chủ ảo) trên AWS Cloud) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `EC2`.

**❌ Đáp án D:**
- Tại sao sai: Đáp án này không trực tiếp giải quyết yêu cầu cụ thể của đề, hoặc dịch vụ/tính năng này có giới hạn so với đáp án đúng.
- Khi nào đúng: Đáp án này sẽ đúng trong bối cảnh khác, nơi ưu tiên khác với yêu cầu của câu hỏi này.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **B** = `EC2`. Scale = Auto Scaling + Load Balancer, thêm/bớt instance theo tải.*
