# Câu hỏi #569

A Cloud Practitioner must determine if any security groups in an AWS account have been provisioned to allow unrestricted access for specific ports. What is the SIMPLEST way to do this?

## Options

**A.** Review the inbound rules for each security group in the Amazon EC2 management console to check for port 0.0.0.0/0.
**B.** Run AWS Trusted Advisor and review the findings.
**C.** Open the AWS IAM console and check the inbound rule filters for open access.
**D.** In AWS Config, create a custom rule that invokes an AWS Lambda function to review firewall rules for inbound access.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Ngữ cảnh đề bài: _A Cloud Practitioner must determine if any security groups in an AWS account have been provisioned to allow unrestricted..._
- **Existing Resources:** Không có tài nguyên cụ thể — câu hỏi thuộc dạng lý thuyết/khái niệm.
- **Current Issue/Goal:** Lựa chọn giải pháp bảo mật tốt nhất theo best practice của AWS.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `Security` | Yêu cầu bảo mật cao |
| `Web application` | Ứng dụng web/API |
| ``EC2`` | dịch vụ compute (máy chủ ảo) trên AWS Cloud |
| ``Lambda`` | dịch vụ serverless compute chạy theo sự kiện |
| ``IAM`` | Identity and Access Management - quản lý quyền truy cập |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Most secure / Security best practice
- **Constraints:** Bảo mật dữ liệu và quyền truy cập

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: B**

**Giải thích:**
Đáp án `Trusted Advisor` đúng vì cố vấn tối ưu chi phí, bảo mật, hiệu năng, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `Trusted Advisor` đóng vai trò: đưa ra khuyến nghị để tối ưu môi trường AWS.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- Tại sao sai: `EC2` (dịch vụ compute (máy chủ ảo) trên AWS Cloud) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu bảo mật chính là chạy ứng dụng, web server, batch processing này.

**❌ Đáp án C:**
- Tại sao sai: `IAM` (Identity and Access Management - quản lý quyền truy cập) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu bảo mật chính là quản lý user, group, role, policy cho AWS resources này.

**❌ Đáp án D:**
- Tại sao sai: `Lambda` (dịch vụ serverless compute chạy theo sự kiện) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu bảo mật chính là chạy code mà không cần quản lý server, trả theo millisecond này.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **B** = `EC2`. Bảo mật = defense-in-depth: nhiều lớp, nguyên tắc least privilege.*
