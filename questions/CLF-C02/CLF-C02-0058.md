# Câu hỏi #58

What is the AWS feature that provides an additional level of security above the default authentication mechanism of usernames and passwords?

## Options

**A.** Encrypted keys.
**B.** Email verification.
**C.** AWS KMS.
**D.** AWS MFA.

Đáp án **D** mô tả đúng khái niệm/dịch vụ mà đề đang hỏi tới.
Đây là đáp án phù hợp nhất với best practice của AWS trong tình huống này.
