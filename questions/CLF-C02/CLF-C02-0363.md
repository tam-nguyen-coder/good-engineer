# Câu hỏi #363

You are working as a site reliability engineer (SRE) in an AWS environment, which of the following services helps monitor your applications?

## Options

**A.** Amazon CloudWatch.
**B.** Amazon CloudSearch.
**C.** Amazon Elastic MapReduce.
**D.** Amazon CloudHSM.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Ngữ cảnh đề bài: _You are working as a site reliability engineer (SRE) in an AWS environment, which of the following services helps monito..._

- **Current Issue/Goal:** Hệ thống cần co giãn linh hoạt theo tải, không bị giới hạn bởi cấu hình cố định.

| Keyword | Ý nghĩa / Gợi ý |

| ``CloudWatch`` | dịch vụ monitoring và observability |

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: A**

**Giải thích:**
Đáp án `CloudWatch` đúng vì dịch vụ monitoring và observability, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `CloudWatch` đóng vai trò: thu thập metric, log, set alarm, dashboard.
