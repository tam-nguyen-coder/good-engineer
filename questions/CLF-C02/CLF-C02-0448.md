# Câu hỏi #448

How would an AWS customer easily apply common access controls to a large set of users?

## Options

**A.** Apply an IAM policy to an IAM group.
**B.** Apply an IAM policy to an IAM role.
**C.** Apply the same IAM policy to all IAM users with access to the same workload.
**D.** Apply an IAM policy to an Amazon Cognito user pool.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Ngữ cảnh đề bài: _How would an AWS customer easily apply common access controls to a large set of users?_
- **Existing Resources:** Không có tài nguyên cụ thể — câu hỏi thuộc dạng lý thuyết/khái niệm.

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: A**

**Giải thích:**
Đáp án `IAM` đúng vì Identity and Access Management - quản lý quyền truy cập, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `IAM` đóng vai trò: quản lý user, group, role, policy cho AWS resources.
