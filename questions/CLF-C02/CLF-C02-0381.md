# Câu hỏi #381

Which of the following AWS services integrates with AWS Shield and AWS Web Application Firewall (AWS WAF) to protect against network and application layer DDoS attacks?

## Options

**A.** Amazon EFS.
**B.** AWS Secrets Manager.
**C.** AWS Systems Manager.
**D.** Amazon CloudFront.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Ngữ cảnh đề bài: _Which of the following AWS services integrates with AWS Shield and AWS Web Application Firewall (AWS WAF) to protect aga..._
- **Existing Resources:** Đề nhắc tới: `WAF` — Web Application Firewall; `Shield` — dịch vụ chống DDoS.
- **Current Issue/Goal:** Lựa chọn giải pháp bảo mật tốt nhất theo best practice của AWS.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `Security` | Yêu cầu bảo mật cao |
| `Web application` | Ứng dụng web/API |
| `DDoS protection` | Chống tấn công từ chối dịch vụ |
| ``CloudFront`` | CDN (Content Delivery Network) toàn cầu |
| ``Secrets Manager`` | dịch vụ lưu trữ và xoay vòng secret |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Most secure / Security best practice
- **Constraints:** Bảo mật dữ liệu và quyền truy cập

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: D**

**Giải thích:**
Đáp án `CloudFront` đúng vì CDN (Content Delivery Network) toàn cầu, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `CloudFront` đóng vai trò: phân phối nội dung tĩnh/động qua edge location, giảm latency.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- Tại sao sai: `EFS` (Elastic File System - file system NFS managed) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu bảo mật chính là chia sẻ file giữa nhiều EC2, Linux, scale tự động này.

**❌ Đáp án B:**
- Tại sao sai: `Secrets Manager` (dịch vụ lưu trữ và xoay vòng secret) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu bảo mật chính là quản lý API key, password, certificate có tự động rotate này.

**❌ Đáp án C:**
- Tại sao sai: `Systems Manager` (quản lý fleet EC2 và on-premises) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu bảo mật chính là patch, automation, parameter store, session manager này.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **D** = `CloudFront`. Bảo mật = defense-in-depth: nhiều lớp, nguyên tắc least privilege.*
