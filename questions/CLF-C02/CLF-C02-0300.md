# Câu hỏi #300

You have a real-time IoT application that requires sub-millisecond latency. Which of the following services should you use?

## Options

**A.** Amazon Redshift.
**B.** Amazon Athena.
**C.** AWS Cloud9.
**D.** Amazon ElastiCache for Redis.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Ngữ cảnh đề bài: _You have a real-time IoT application that requires sub-millisecond latency. Which of the following services should you u..._
- **Existing Resources:** Không có tài nguyên cụ thể — câu hỏi thuộc dạng lý thuyết/khái niệm.
- **Current Issue/Goal:** Lựa chọn giải pháp có hiệu năng cao nhất, độ trễ thấp nhất.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `Real-time` | Xử lý thời gian thực |
| `IoT` | Thu thập dữ liệu từ thiết bị |
| ``Athena`` | query S3 bằng SQL |
| ``Redshift`` | data warehouse columnar |
| ``ElastiCache`` | in-memory cache (Redis, Memcached) |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Best performance / Lowest latency
- **Constraints:** Hiệu năng và độ trễ thấp

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: D**

**Giải thích:**
Đáp án `ElastiCache` đúng vì in-memory cache (Redis, Memcached), phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `ElastiCache` đóng vai trò: tăng tốc ứng dụng bằng cache, giảm tải database.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- Tại sao sai: `Redshift` (data warehouse columnar) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu phân tích dữ liệu lớn, OLAP, BI được ưu tiên hơn hiệu năng.

**❌ Đáp án B:**
- Tại sao sai: `Athena` (query S3 bằng SQL) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu phân tích dữ liệu trong S3 mà không cần copy được ưu tiên hơn hiệu năng.

**❌ Đáp án C:**
- Tại sao sai: Đáp án này không trực tiếp giải quyết yêu cầu cụ thể của đề, hoặc dịch vụ/tính năng này có giới hạn so với đáp án đúng.
- Khi nào đúng: Đáp án này sẽ đúng trong bối cảnh khác, nơi ưu tiên khác với yêu cầu của câu hỏi này.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **D** = `Athena`. Hiệu năng = edge location + cache + chọn instance/region gần user.*
