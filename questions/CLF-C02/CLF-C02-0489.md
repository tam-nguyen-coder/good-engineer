# Câu hỏi #489

Which AWS services can host a Microsoft SQL Server database? (Select TWO)

## Options

**A.** Amazon EC2.
**B.** Amazon Relational Database Service (Amazon RDS).
**C.** Amazon Aurora.
**D.** Amazon Redshift.
**E.** Amazon S3.

| ``EC2`` | dịch vụ compute (máy chủ ảo) trên AWS Cloud |
