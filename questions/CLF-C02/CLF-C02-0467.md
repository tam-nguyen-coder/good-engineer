# Câu hỏi #467

Which of the following is an AWS managed Domain Name System (DNS) web service?

## Options

**A.** Amazon Route 53.
**B.** Amazon Neptune.
**C.** Amazon SageMaker.
**D.** Amazon Lightsail.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Đề hỏi về khái niệm/dịch vụ AWS liên quan tới: _Which of the following is an AWS managed Domain Name System (DNS) web service?_
- **Existing Resources:** Đề nhắc tới: `Route 53` — dịch vụ DNS và domain registration.
- **Current Issue/Goal:** Xác định đúng khái niệm/dịch vụ/tính năng AWS được hỏi tới.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| ``Route 53`` | dịch vụ DNS và domain registration |
| ``Neptune`` | graph database managed |
| ``SageMaker`` | ML platform managed |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Networking / DNS / CDN
- **Constraints:** Tuân theo best practice AWS

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: A**

**Giải thích:**
Đáp án `Route 53` đúng vì dịch vụ DNS và domain registration, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `Route 53` đóng vai trò: phân giải tên miền, health check, routing policy.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án B:**
- Tại sao sai: `Neptune` (graph database managed) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `Neptune`.

**❌ Đáp án C:**
- Tại sao sai: `SageMaker` (ML platform managed) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `SageMaker`.

**❌ Đáp án D:**
- Tại sao sai: Đáp án này không trực tiếp giải quyết yêu cầu cụ thể của đề, hoặc dịch vụ/tính năng này có giới hạn so với đáp án đúng.
- Khi nào đúng: Đáp án này sẽ đúng trong bối cảnh khác, nơi ưu tiên khác với yêu cầu của câu hỏi này.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **A** = `Route 53` — phù hợp với yêu cầu đề bài.*
