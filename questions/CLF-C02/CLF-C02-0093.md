# Câu hỏi #93

What is the AWS service/feature that takes advantage of Amazon CloudFront's globally distributed edge locations to transfer files to S3 with higher upload speeds?

## Options

**A.** S3 Transfer Acceleration.
**B.** AWS WAF.
**C.** AWS Snowmobile.
**D.** AWS Snowball.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Ngữ cảnh đề bài: _What is the AWS service/feature that takes advantage of Amazon CloudFront's globally distributed edge locations to trans..._
- **Existing Resources:** Đề nhắc tới: `S3` — dịch vụ lưu trữ object bền vững (11x 9s); `CloudFront` — CDN (Content Delivery Network) toàn cầu.
- **Current Issue/Goal:** Lựa chọn giải pháp có hiệu năng cao nhất, độ trễ thấp nhất.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `Global / Multi-region` | Phạm vi toàn cầu, nhiều region |
| ``S3`` | dịch vụ lưu trữ object bền vững (11x 9s) |
| ``CloudFront`` | CDN (Content Delivery Network) toàn cầu |
| ``WAF`` | Web Application Firewall |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Best performance / Lowest latency
- **Constraints:** Hiệu năng và độ trễ thấp

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: A**

**Giải thích:**
Đáp án `S3` đúng vì dịch vụ lưu trữ object bền vững (11x 9s), phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `S3` đóng vai trò: lưu trữ file, backup, static website, data lake, log archive.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án B:**
- Tại sao sai: `WAF` (Web Application Firewall) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu lọc traffic web ngăn chặn SQL injection, XSS, bot được ưu tiên hơn hiệu năng.

**❌ Đáp án C:**
- Tại sao sai: `Snowball/Snowmobile` (thiết bị vật lý chuyển dữ liệu quy mô lớn) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu di chuyển petabyte dữ liệu khi bandwidth hạn chế được ưu tiên hơn hiệu năng.

**❌ Đáp án D:**
- Tại sao sai: `Snowball/Snowmobile` (thiết bị vật lý chuyển dữ liệu quy mô lớn) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu di chuyển petabyte dữ liệu khi bandwidth hạn chế được ưu tiên hơn hiệu năng.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **A** = `S3`. Hiệu năng = edge location + cache + chọn instance/region gần user.*
