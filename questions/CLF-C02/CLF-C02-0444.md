# Câu hỏi #444

Which statement best describes Elastic Load Balancing?

## Options

**A.** It translates a domain name into an IP address using DNC.
**B.** It distributes incoming application traffic across one or more Amazon EC2 instances.
**C.** It collects metrics on connected Amazon EC2 instances.
**D.** It automatically adjusts the number of Amazon EC2 instances to support incoming traffic.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Đề hỏi về khái niệm/dịch vụ AWS liên quan tới: _Which statement best describes Elastic Load Balancing?_
- **Existing Resources:** Đề nhắc tới: `Elastic Load Balancer` — cân bằng tải (ALB, NLB, GLB).
- **Current Issue/Goal:** Hệ thống cần co giãn linh hoạt theo tải, không bị giới hạn bởi cấu hình cố định.

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: B**

**Giải thích:**
Đáp án `EC2` đúng vì dịch vụ compute (máy chủ ảo) trên AWS Cloud, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `EC2` đóng vai trò: chạy ứng dụng, web server, batch processing.
