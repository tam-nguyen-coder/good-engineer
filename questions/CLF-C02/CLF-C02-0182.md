# Câu hỏi #182

Which of the following is NOT a characteristic of Amazon Elastic Compute Cloud (Amazon EC2)?

## Options

**A.** Amazon EC2 is considered a Serverless Web Service.
**B.** Amazon EC2 eliminates the need to invest in hardware upfront.
**C.** Amazon EC2 can launch as many or as few virtual servers as needed.
**D.** Amazon EC2 offers scalable computing.


---

- **Current Issue/Goal:** Hệ thống cần co giãn linh hoạt theo tải, không bị giới hạn bởi cấu hình cố định.

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: A**

**Giải thích:**
Đáp án `EC2` đúng vì dịch vụ compute (máy chủ ảo) trên AWS Cloud, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `EC2` đóng vai trò: chạy ứng dụng, web server, batch processing.
