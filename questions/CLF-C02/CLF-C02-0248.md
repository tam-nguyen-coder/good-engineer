# Câu hỏi #248

Availability Zones within a Region are connected over low-latency links. Which of the following is a benefit of these links?

## Options

**A.** Create private connection to your data center.
**B.** Achieve global high availability.
**C.** Automate the process of provisioning new compute resources.
**D.** Make synchronous replication of your data possible.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Ngữ cảnh đề bài: _Availability Zones within a Region are connected over low-latency links. Which of the following is a benefit of these li..._
- **Existing Resources:** Không có tài nguyên cụ thể — câu hỏi thuộc dạng lý thuyết/khái niệm.
- **Current Issue/Goal:** Lựa chọn giải pháp có hiệu năng cao nhất, độ trễ thấp nhất.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `Highly available` | Yêu cầu hệ thống không bị gián đoạn |
| `Global / Multi-region` | Phạm vi toàn cầu, nhiều region |
| `Hybrid / On-premises` | Kết nối môi trường on-premises với AWS |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Best performance / Lowest latency
- **Constraints:** Hiệu năng và độ trễ thấp

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: D**

**Giải thích:**
Đáp án **D** mô tả đúng khái niệm/dịch vụ mà đề đang hỏi tới.
Đây là đáp án phù hợp nhất với best practice của AWS trong tình huống này.
