# Câu hỏi #481

Which of the following services is in the category of AWS serverless platform?

## Options

**A.** Amazon EMR.
**B.** Elastic Load Balancing.
**C.** AWS Lambda.
**D.** AWS Mobile Hub.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Đề hỏi về khái niệm/dịch vụ AWS liên quan tới: _Which of the following services is in the category of AWS serverless platform?_
- **Existing Resources:** Không có tài nguyên cụ thể — câu hỏi thuộc dạng lý thuyết/khái niệm.
- **Current Issue/Goal:** Hệ thống cần co giãn linh hoạt theo tải, không bị giới hạn bởi cấu hình cố định.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `Serverless` | Không muốn quản lý server |
| ``Lambda`` | dịch vụ serverless compute chạy theo sự kiện |
| ``Elastic Load Balancer`` | cân bằng tải (ALB, NLB, GLB) |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Scalability / Elasticity
- **Constraints:** Khả năng mở rộng theo tải

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: C**

**Giải thích:**
Đáp án `Lambda` đúng vì dịch vụ serverless compute chạy theo sự kiện, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `Lambda` đóng vai trò: chạy code mà không cần quản lý server, trả theo millisecond.

- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `Elastic Load Balancer`.
