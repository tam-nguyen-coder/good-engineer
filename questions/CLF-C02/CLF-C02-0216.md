# Câu hỏi #216

You are running a financial services web application on AWS. The application uses a MySQL database to store the data. Which of the following AWS services would improve the performance of your application by allowing you to retrieve information from fast in-memory caches?

## Options

**A.** Amazon EFS.
**B.** Amazon Neptune.
**C.** Amazon ElastiCache.
**D.** DAX.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Ngữ cảnh đề bài: _You are running a financial services web application on AWS. The application uses a MySQL database to store the data. Wh..._
- **Existing Resources:** Không có tài nguyên cụ thể — câu hỏi thuộc dạng lý thuyết/khái niệm.
- **Current Issue/Goal:** Lựa chọn giải pháp có hiệu năng cao nhất, độ trễ thấp nhất.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `Performance` | Yêu cầu hiệu năng, độ trễ thấp |
| `Web application` | Ứng dụng web/API |
| ``EFS`` | Elastic File System - file system NFS managed |
| ``Neptune`` | graph database managed |
| ``ElastiCache`` | in-memory cache (Redis, Memcached) |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Best performance / Lowest latency
- **Constraints:** Hiệu năng và độ trễ thấp

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: C**

**Giải thích:**
Đáp án `ElastiCache` đúng vì in-memory cache (Redis, Memcached), phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `ElastiCache` đóng vai trò: tăng tốc ứng dụng bằng cache, giảm tải database.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- Tại sao sai: `EFS` (Elastic File System - file system NFS managed) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu chia sẻ file giữa nhiều EC2, Linux, scale tự động được ưu tiên hơn hiệu năng.

**❌ Đáp án B:**
- Tại sao sai: `Neptune` (graph database managed) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu lưu trữ và truy vấn quan hệ đồ thị được ưu tiên hơn hiệu năng.

**❌ Đáp án D:**
- Tại sao sai: Đáp án này không trực tiếp giải quyết yêu cầu cụ thể của đề, hoặc dịch vụ/tính năng này có giới hạn so với đáp án đúng.
- Khi nào đúng: Đáp án này sẽ đúng trong bối cảnh khác, nơi ưu tiên khác với yêu cầu của câu hỏi này.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **C** = `EFS`. Hiệu năng = edge location + cache + chọn instance/region gần user.*
