# Câu hỏi #356

You have just finished writing your application code. Which service can be used to automate the deployment and scaling of your application?

## Options

**A.** Amazon Simple Storage Service.
**B.** AWS Elastic Beanstalk.
**C.** AWS CodeCommit.
**D.** Amazon Elastic File System.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Ngữ cảnh đề bài: _You have just finished writing your application code. Which service can be used to automate the deployment and scaling o..._
- **Existing Resources:** Không có tài nguyên cụ thể — câu hỏi thuộc dạng lý thuyết/khái niệm.
- **Current Issue/Goal:** Hệ thống cần co giãn linh hoạt theo tải, không bị giới hạn bởi cấu hình cố định.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| ``S3`` | dịch vụ lưu trữ object bền vững (11x 9s) |
| ``Elastic Beanstalk`` | PaaS - nền tảng triển khai ứng dụng |
| ``EFS`` | Elastic File System - file system NFS managed |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Scalability / Elasticity
- **Constraints:** Khả năng mở rộng theo tải

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: B**

**Giải thích:**
Đáp án `Elastic Beanstalk` đúng vì PaaS - nền tảng triển khai ứng dụng, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `Elastic Beanstalk` đóng vai trò: deploy code mà AWS tự lo capacity balancing, scaling.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- Tại sao sai: `S3` (dịch vụ lưu trữ object bền vững (11x 9s)) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `S3`.

**❌ Đáp án C:**
- Tại sao sai: Đáp án này không trực tiếp giải quyết yêu cầu cụ thể của đề, hoặc dịch vụ/tính năng này có giới hạn so với đáp án đúng.
- Khi nào đúng: Đáp án này sẽ đúng trong bối cảnh khác, nơi ưu tiên khác với yêu cầu của câu hỏi này.

**❌ Đáp án D:**
- Tại sao sai: `EFS` (Elastic File System - file system NFS managed) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `EFS`.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **B** = `S3`. Scale = Auto Scaling + Load Balancer, thêm/bớt instance theo tải.*
