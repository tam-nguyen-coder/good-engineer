# Câu hỏi #592

A company is considering using AWS for a self-hosted database that requires a nightly shutdown for maintenance and cost-saving purposes. Which service should the company use?

## Options

**A.** Amazon Redshift.
**B.** Amazon DynamoDB.
**C.** Amazon Elastic Compute Cloud (Amazon EC2) with Amazon EC2 instance store.
**D.** Amazon EC2 with Amazon Elastic Block Store (Amazon EBS).


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Tình huống được mô tả: _A company is considering using AWS for a self-hosted database that requires a nightly shutdown for maintenance and cost-..._
- **Existing Resources:** Không có tài nguyên cụ thể — câu hỏi thuộc dạng lý thuyết/khái niệm.
- **Current Issue/Goal:** Lựa chọn giải pháp giúp tối ưu/tiết kiệm chi phí nhất mà vẫn đáp ứng yêu cầu nghiệp vụ.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| ``EC2`` | dịch vụ compute (máy chủ ảo) trên AWS Cloud |
| ``DynamoDB`` | dịch vụ NoSQL key-value được quản lý hoàn toàn |
| ``EBS`` | Elastic Block Store - ổ đĩa block cho EC2 |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Most cost-effective / Cost optimization
- **Constraints:** Ngân sách/chi phí là yếu tố quan trọng

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: D**

**Giải thích:**
Đáp án `EC2` đúng vì dịch vụ compute (máy chủ ảo) trên AWS Cloud, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `EC2` đóng vai trò: chạy ứng dụng, web server, batch processing.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- Tại sao sai: `Redshift` (data warehouse columnar) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi ưu tiên phân tích dữ liệu lớn, OLAP, BI mà không yêu cầu tối ưu chi phí.

**❌ Đáp án B:**
- Tại sao sai: `DynamoDB` (dịch vụ NoSQL key-value được quản lý hoàn toàn) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi ưu tiên lưu trữ dữ liệu phi quan hệ với độ trễ thấp ở mọi quy mô mà không yêu cầu tối ưu chi phí.

**❌ Đáp án C:**
- Tại sao sai: `EC2` (dịch vụ compute (máy chủ ảo) trên AWS Cloud) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi ưu tiên chạy ứng dụng, web server, batch processing mà không yêu cầu tối ưu chi phí.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **D** = `EC2`. Tiết kiệm = pay-as-you-go, đừng mua hạ tầng thừa.*
