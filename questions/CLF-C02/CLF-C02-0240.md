# Câu hỏi #240

Which of the following will help AWS customers save on costs when migrating their workloads to AWS?

## Options

**A.** Use servers instead of managed services.
**B.** Use existing third-party software licenses on AWS.
**C.** Migrate production workloads to AWS edge locations instead of AWS Regions.
**D.** Use AWS Outposts to run all workloads in a cost-optimized environment.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Đề hỏi về khái niệm/dịch vụ AWS liên quan tới: _Which of the following will help AWS customers save on costs when migrating thei..._
- **Existing Resources:** Không có tài nguyên cụ thể — câu hỏi thuộc dạng lý thuyết/khái niệm.
- **Current Issue/Goal:** Lựa chọn giải pháp giúp tối ưu/tiết kiệm chi phí nhất mà vẫn đáp ứng yêu cầu nghiệp vụ.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| ``Outposts`` | AWS hardware chạy trong on-premises |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Most cost-effective / Cost optimization
- **Constraints:** Ngân sách/chi phí là yếu tố quan trọng

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: B**

**Giải thích:**
Đáp án **B** mô tả đúng khái niệm/dịch vụ mà đề đang hỏi tới.
Đây là đáp án phù hợp nhất với best practice của AWS trong tình huống này.

- Tại sao sai: Đáp án này không trực tiếp giải quyết yêu cầu cụ thể của đề, hoặc dịch vụ/tính năng này có giới hạn so với đáp án đúng.
- Khi nào đúng: Đáp án này sẽ đúng trong bối cảnh khác, nơi ưu tiên khác với yêu cầu của câu hỏi này.
