# Câu hỏi #456

Which of the following is the customer's responsibility under the AWS shared responsibility model?

## Options

**A.** Patching underlying infrastructure
**B.** Physical security
**C.** Patching Amazon EC2 instances
**D.** Patching network infrastructure


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Ngữ cảnh đề bài: _Which of the following is the customer's responsibility under the AWS shared responsibility model?_
- **Existing Resources:** Đề nhắc tới: `Shared Responsibility Model` — mô hình trách nhiệm chung.
- **Current Issue/Goal:** Lựa chọn giải pháp bảo mật tốt nhất theo best practice của AWS.

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: C**

**Giải thích:**
Đáp án `EC2` đúng vì dịch vụ compute (máy chủ ảo) trên AWS Cloud, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `EC2` đóng vai trò: chạy ứng dụng, web server, batch processing.
