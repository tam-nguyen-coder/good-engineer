# Câu hỏi #429

Which service provides a virtually unlimited amount of online highly durable object storage?

## Options

**A.** Amazon Redshift.
**B.** Amazon Elastic File System (Amazon EFS).
**C.** Amazon Elastic Container Service (Amazon ECS).
**D.** Amazon S3.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Câu hỏi về dịch vụ cung cấp object storage với dung lượng gần như vô hạn và độ bền cao.
- **Existing Resources:** Không có cụ thể — bài toán khái niệm.
- **Current Issue/Goal:** Xác định dịch vụ object storage scale lớn, durable.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `virtually unlimited` | Dung lượng không giới hạn thực tế |
| `highly durable` | Độ bền dữ liệu 99.999999999% (11 chín 9) |
| `object storage` | Lưu trữ dạng object (key-value) |
| `Amazon S3` | Object storage scale vô hạn, durability 11 chín 9 |
| `Amazon Redshift` | Data warehouse, không phải object storage |
| `Amazon EFS` | NFS file system, có giới hạn throughput/dung lượng |
| `Amazon ECS` | Container orchestration, không phải storage |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Xác định dịch vụ storage
- **Constraints:** Phải là object storage, scale vô hạn, highly durable

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: D**

**Giải thích:**
- `Amazon S3` (Simple Storage Service) là object storage scale lên đến hàng tỷ object, tổng dung lượng không giới hạn thực tế.
- S3 Standard cung cấp durability 99.999999999% (11 chín 9) — tức là nếu lưu 10 triệu object, trung bình chỉ mất 1 object mỗi 10.000 năm.
- Đây là dịch vụ phù hợp cho data lake, backup, archive, static website, big data analytics.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- `Amazon Redshift` là cloud data warehouse (columnar MPP database), không phải object storage.
- Đáp án này dùng cho OLAP và BI queries.

**❌ Đáp án B:**
- `Amazon EFS` là NFS file system cho Linux. Có giới hạn throughput (theo size của file system) và dung lượng tối đa (hiện tại petabyte-scale nhưng cần cấu hình performance mode).
- Đáp án này dùng cho shared filesystem giữa nhiều EC2.

**❌ Đáp án C:**
- `Amazon ECS` (Elastic Container Service) là container orchestration service, không phải storage service.
- Đáp án này dùng để chạy Docker containers ở quy mô lớn.

## 6. MẺO GHI NHỚ (Memory Hook)
🧠 *Object storage vô hạn + 11 chín 9 = `S3`. File system shared = `EFS`. Block storage = `EBS`. Data warehouse = `Redshift`.*
