# Câu hỏi #26

Hundreds of thousands of DDoS attacks are recorded every month worldwide. What service does AWS provide to help protect AWS Customers from these attacks? (Choose TWO)

## Options

**A.** AWS Shield.
**B.** AWS Config.
**C.** Amazon Cognito.
**D.** AWS WAF.
**E.** AWS KMS.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Có hàng trăm nghìn cuộc tấn công `DDoS` mỗi tháng trên toàn cầu. Cần chọn dịch vụ AWS giúp bảo vệ khách hàng khỏi các cuộc tấn công này.
- **Existing Resources:** Web application/app trên AWS.
- **Current Issue/Goal:** Bảo vệ khỏi DDoS và các cuộc tấn công web.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `DDoS` | Tấn công từ chối dịch vụ phân tán, làm ngập traffic |
| `AWS Shield` | Dịch vụ bảo vệ DDoS chuyên dụng (Standard miễn phí, Advanced trả phí) |
| `AWS WAF` | Web Application Firewall — lọc request HTTP theo rule |
| `AWS Config` | Theo dõi config resource, không phải security perimeter |
| `Amazon Cognito` | Quản lý identity/user cho app, không chống DDoS |
| `AWS KMS` | Key Management Service — mã hóa, không chống DDoS |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Multi-answer (Choose TWO) — Security services chống DDoS
- **Constraints:** Phải là service bảo vệ khỏi DDoS hoặc web attack

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: A và D**

**Giải thích:**
- **A — AWS Shield:** Dịch vụ bảo vệ DDoS chuyên dụng của AWS. `Shield Standard` miễn phí, tự động bảo vệ mọi AWS customer khỏi các cuộc tấn công L3/L4 phổ biến (SYN flood, UDP flood, ...). `Shield Advanced` (trả phí) cung cấp thêm detection thông minh, real-time visibility và bảo vệ cho `Route 53`, `CloudFront`, `ALB`, `ELB`, ...
- **D — AWS WAF:** Web Application Firewall giúp lọc và ngăn chặn các cuộc tấn công web phổ biến (SQL injection, XSS, bot traffic) thông qua các rule tùy chỉnh hoặc `Managed Rules`. Kết hợp với Shield Advanced tạo lớp bảo vệ toàn diện.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án B:**
- `AWS Config` theo dõi và audit thay đổi cấu hình resource, không có khả năng ngăn chặn DDoS.

**❌ Đáp án C:**
- `Amazon Cognito` cung cấp authentication/authorization cho user (identity pool, user pool), không phải tường lửa chống DDoS.

**❌ Đáp án E:**
- `AWS KMS` cung cấp key để mã hóa dữ liệu, không có chức năng lọc traffic hay chống DDoS.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *"Chống DDoS = `AWS Shield` (mạng) + `AWS WAF` (web layer). `KMS` = mã hóa, `Cognito` = identity, `Config` = config tracking."*
