# Câu hỏi #270

Which of the following is a feature of Amazon RDS that performs automatic failover when the primary database fails to respond?

## Options

**A.** RDS Single-AZ.
**B.** RDS Write Replica.
**C.** RDS Snapshots.
**D.** RDS Multi-AZ.
