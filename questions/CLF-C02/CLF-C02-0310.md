# Câu hỏi #310

Which AWS service collects metrics from running EC2 instances?

## Options

**A.** Amazon Inspector.
**B.** Amazon CloudWatch.
**C.** AWS CloudFormation.
**D.** AWS CloudTrail.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Đề hỏi về khái niệm/dịch vụ AWS liên quan tới: _Which AWS service collects metrics from running EC2 instances?_
- **Existing Resources:** Đề nhắc tới: `EC2` — dịch vụ compute (máy chủ ảo) trên AWS Cloud.
- **Current Issue/Goal:** Cần giám sát, theo dõi và ghi nhận hoạt động của hệ thống/tài khoản.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| ``EC2`` | dịch vụ compute (máy chủ ảo) trên AWS Cloud |
| ``CloudWatch`` | dịch vụ monitoring và observability |
| ``CloudTrail`` | dịch vụ audit API call |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Monitoring / Auditing
- **Constraints:** Tuân theo best practice AWS

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: B**

**Giải thích:**
Đáp án `CloudWatch` đúng vì dịch vụ monitoring và observability, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `CloudWatch` đóng vai trò: thu thập metric, log, set alarm, dashboard.

- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `CloudFormation`.

**❌ Đáp án D:**
- Tại sao sai: `CloudTrail` (dịch vụ audit API call) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `CloudTrail`.
