# Câu hỏi #211

A company has a web application that is hosted on a single EC2 instance and is approaching 100 percent CPU Utilization during peak loads. Rather than scaling the server vertically, the company has decided to deploy three Amazon EC2 instances in parallel and to distribute traffic across the three servers. What AWS Service should the company use to distribute the traffic evenly?

## Options

**A.** AWS Global Accelerator.
**B.** AWS Application Load Balancer (ALB).
**C.** Amazon CloudFront.
**D.** Transit VPC.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Tình huống được mô tả: _A company has a web application that is hosted on a single EC2 instance and is approaching 100 percent CPU Utilization d..._
- **Existing Resources:** Đề nhắc tới: `EC2` — dịch vụ compute (máy chủ ảo) trên AWS Cloud.
- **Current Issue/Goal:** Hệ thống cần co giãn linh hoạt theo tải, không bị giới hạn bởi cấu hình cố định.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `Global / Multi-region` | Phạm vi toàn cầu, nhiều region |
| `Web application` | Ứng dụng web/API |
| ``EC2`` | dịch vụ compute (máy chủ ảo) trên AWS Cloud |
| ``VPC`` | Virtual Private Cloud - mạng riêng ảo trong AWS |
| ``CloudFront`` | CDN (Content Delivery Network) toàn cầu |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Scalability / Elasticity
- **Constraints:** Khả năng mở rộng theo tải

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: B**

**Giải thích:**
Đáp án `Elastic Load Balancer` đúng vì cân bằng tải (ALB, NLB, GLB), phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `Elastic Load Balancer` đóng vai trò: phân phối traffic tới nhiều target, health check.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- Tại sao sai: Đáp án này không trực tiếp giải quyết yêu cầu cụ thể của đề, hoặc dịch vụ/tính năng này có giới hạn so với đáp án đúng.
- Khi nào đúng: Đáp án này sẽ đúng trong bối cảnh khác, nơi ưu tiên khác với yêu cầu của câu hỏi này.

**❌ Đáp án C:**
- Tại sao sai: `CloudFront` (CDN (Content Delivery Network) toàn cầu) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `CloudFront`.

**❌ Đáp án D:**
- Tại sao sai: `VPC` (Virtual Private Cloud - mạng riêng ảo trong AWS) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `VPC`.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **B** = `EC2`. Scale = Auto Scaling + Load Balancer, thêm/bớt instance theo tải.*
