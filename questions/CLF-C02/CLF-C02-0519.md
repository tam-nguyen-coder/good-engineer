# Câu hỏi #519

Which AWS service provides a simple and scalable shared file storage solution for use with Linux-based AWS and on-premises servers?

## Options

**A.** Amazon S3.
**B.** Amazon Glacier.
**C.** Amazon EBS.
**D.** Amazon EFS.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Ngữ cảnh đề bài: _Which AWS service provides a simple and scalable shared file storage solution for use with Linux-based AWS and on-premis..._
- **Existing Resources:** Không có tài nguyên cụ thể — câu hỏi thuộc dạng lý thuyết/khái niệm.
- **Current Issue/Goal:** Hệ thống cần co giãn linh hoạt theo tải, không bị giới hạn bởi cấu hình cố định.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `Hybrid / On-premises` | Kết nối môi trường on-premises với AWS |
| ``S3`` | dịch vụ lưu trữ object bền vững (11x 9s) |
| ``Glacier`` | dịch vụ lưu trữ lạnh giá rẻ |
| ``EBS`` | Elastic Block Store - ổ đĩa block cho EC2 |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Scalability / Elasticity
- **Constraints:** Khả năng mở rộng theo tải

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: D**

**Giải thích:**
Đáp án `EFS` đúng vì Elastic File System - file system NFS managed, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `EFS` đóng vai trò: chia sẻ file giữa nhiều EC2, Linux, scale tự động.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- Tại sao sai: `S3` (dịch vụ lưu trữ object bền vững (11x 9s)) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `S3`.

**❌ Đáp án B:**
- Tại sao sai: `Glacier` (dịch vụ lưu trữ lạnh giá rẻ) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `Glacier`.

**❌ Đáp án C:**
- Tại sao sai: `EBS` (Elastic Block Store - ổ đĩa block cho EC2) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `EBS`.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **D** = `S3`. Scale = Auto Scaling + Load Balancer, thêm/bớt instance theo tải.*
