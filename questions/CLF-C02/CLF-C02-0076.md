# Câu hỏi #76

You are working on two projects that require completely different network configurations. Which AWS service or feature will allow you to isolate resources and network configurations?

## Options

**A.** Internet gateways.
**B.** Virtual Private Cloud.
**C.** Security Groups.
**D.** Amazon CloudFront.

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: B**

**Giải thích:**
Đáp án `VPC` đúng vì Virtual Private Cloud - mạng riêng ảo trong AWS, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `VPC` đóng vai trò: cô lập tài nguyên mạng, kiểm soát subnet, routing, security group.
