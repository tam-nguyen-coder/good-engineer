# Câu hỏi #185

What is the framework created by AWS Professional Services that helps organizations design a road map to successful cloud adoption?

## Options

**A.** AWS Secrets Manager.
**B.** AWS WAF.
**C.** AWS CAF.
**D.** Amazon EFS.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Tình huống được mô tả: _What is the framework created by AWS Professional Services that helps organizations design a road map to successful clou..._
- **Existing Resources:** Đề nhắc tới: `Organizations` — quản lý nhiều tài khoản AWS; `Professional Services` — dịch vụ tư vấn từ chuyên gia AWS.
- **Current Issue/Goal:** Lựa chọn giải pháp bảo mật tốt nhất theo best practice của AWS.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| ``Secrets Manager`` | dịch vụ lưu trữ và xoay vòng secret |
| ``WAF`` | Web Application Firewall |
| ``EFS`` | Elastic File System - file system NFS managed |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Most secure / Security best practice
- **Constraints:** Bảo mật dữ liệu và quyền truy cập

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: C**

**Giải thích:**
Đáp án **C** mô tả đúng khái niệm/dịch vụ mà đề đang hỏi tới.
Đây là đáp án phù hợp nhất với best practice của AWS trong tình huống này.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- Tại sao sai: `Secrets Manager` (dịch vụ lưu trữ và xoay vòng secret) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu bảo mật chính là quản lý API key, password, certificate có tự động rotate này.

**❌ Đáp án B:**
- Tại sao sai: `WAF` (Web Application Firewall) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu bảo mật chính là lọc traffic web ngăn chặn SQL injection, XSS, bot này.

**❌ Đáp án D:**
- Tại sao sai: `EFS` (Elastic File System - file system NFS managed) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu bảo mật chính là chia sẻ file giữa nhiều EC2, Linux, scale tự động này.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **C** = `Secrets Manager`. Bảo mật = defense-in-depth: nhiều lớp, nguyên tắc least privilege.*
