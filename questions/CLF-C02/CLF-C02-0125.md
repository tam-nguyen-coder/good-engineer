# Câu hỏi #125

A company has deployed a new web application on multiple Amazon EC2 instances. Which of the following should they use to ensure that the incoming HTTP traffic is distributed evenly across the instances?

## Options

**A.** AWS EC2 Auto Recovery.
**B.** AWS Auto Scaling.
**C.** AWS Network Load Balancer.
**D.** AWS Application Load Balancer.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Tình huống được mô tả: _A company has deployed a new web application on multiple Amazon EC2 instances. Which of the following should they use to..._
- **Existing Resources:** Đề nhắc tới: `EC2` — dịch vụ compute (máy chủ ảo) trên AWS Cloud.
- **Current Issue/Goal:** Hệ thống cần co giãn linh hoạt theo tải, không bị giới hạn bởi cấu hình cố định.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `Web application` | Ứng dụng web/API |
| ``EC2`` | dịch vụ compute (máy chủ ảo) trên AWS Cloud |
| ``Auto Scaling`` | tự động scale số lượng EC2 |
| ``Elastic Load Balancer`` | cân bằng tải (ALB, NLB, GLB) |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Scalability / Elasticity
- **Constraints:** Khả năng mở rộng theo tải

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: D**

**Giải thích:**
Đáp án `Elastic Load Balancer` đúng vì cân bằng tải (ALB, NLB, GLB), phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `Elastic Load Balancer` đóng vai trò: phân phối traffic tới nhiều target, health check.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- Tại sao sai: `EC2` (dịch vụ compute (máy chủ ảo) trên AWS Cloud) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `EC2`.

**❌ Đáp án B:**
- Tại sao sai: `Auto Scaling` (tự động scale số lượng EC2) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `Auto Scaling`.

**❌ Đáp án C:**
- Tại sao sai: `Elastic Load Balancer` (cân bằng tải (ALB, NLB, GLB)) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `Elastic Load Balancer`.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **D** = `EC2`. Scale = Auto Scaling + Load Balancer, thêm/bớt instance theo tải.*
