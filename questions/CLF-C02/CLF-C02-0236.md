# Câu hỏi #236

Which of the following services can be used to monitor the HTTP and HTTPS requests that are forwarded to Amazon CloudFront?

## Options

**A.** AWS WAF.
**B.** Amazon CloudWatch.
**C.** AWS Cloud9.
**D.** AWS CloudTrail.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Ngữ cảnh đề bài: _Which of the following services can be used to monitor the HTTP and HTTPS requests that are forwarded to Amazon CloudFro..._
- **Existing Resources:** Đề nhắc tới: `CloudFront` — CDN (Content Delivery Network) toàn cầu.
- **Current Issue/Goal:** Cần giám sát, theo dõi và ghi nhận hoạt động của hệ thống/tài khoản.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `Web application` | Ứng dụng web/API |
| ``CloudWatch`` | dịch vụ monitoring và observability |
| ``CloudTrail`` | dịch vụ audit API call |
| ``CloudFront`` | CDN (Content Delivery Network) toàn cầu |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Monitoring / Auditing
- **Constraints:** Tuân theo best practice AWS

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: A**

**Giải thích:**
Đáp án `WAF` đúng vì Web Application Firewall, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `WAF` đóng vai trò: lọc traffic web ngăn chặn SQL injection, XSS, bot.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án B:**
- Tại sao sai: `CloudWatch` (dịch vụ monitoring và observability) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `CloudWatch`.

**❌ Đáp án C:**
- Tại sao sai: Đáp án này không trực tiếp giải quyết yêu cầu cụ thể của đề, hoặc dịch vụ/tính năng này có giới hạn so với đáp án đúng.
- Khi nào đúng: Đáp án này sẽ đúng trong bối cảnh khác, nơi ưu tiên khác với yêu cầu của câu hỏi này.

**❌ Đáp án D:**
- Tại sao sai: `CloudTrail` (dịch vụ audit API call) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `CloudTrail`.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **A** = `CloudWatch` — phù hợp với yêu cầu đề bài.*
