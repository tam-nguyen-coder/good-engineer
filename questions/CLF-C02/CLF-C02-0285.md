# Câu hỏi #285

Select the services that are server-based: (Choose TWO)

## Options

**A.** Amazon RDS.
**B.** Amazon DynamoDB.
**C.** AWS Lambda.
**D.** AWS Fargate.
**E.** Amazon EMR.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Đề hỏi về khái niệm/dịch vụ AWS liên quan tới: _Select the services that are server-based: (Choose TWO)_
- **Existing Resources:** Không có tài nguyên cụ thể — câu hỏi thuộc dạng lý thuyết/khái niệm.
- **Current Issue/Goal:** Lựa chọn đáp án đúng nhất theo yêu cầu của đề.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| ``RDS`` | dịch vụ database quan hệ được quản lý |
| ``DynamoDB`` | dịch vụ NoSQL key-value được quản lý hoàn toàn |
| ``Lambda`` | dịch vụ serverless compute chạy theo sự kiện |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Database
- **Constraints:** Tuân theo best practice AWS

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: A và E**

**Giải thích:**
Đề yêu cầu chọn nhiều đáp án. Hai phương án **A** và **E** cùng đáp ứng các tiêu chí của câu hỏi.
- **A** — `RDS`: dịch vụ database quan hệ được quản lý.
- **E** — Amazon EMR..

- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `Lambda`.
