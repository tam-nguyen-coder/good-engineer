# Câu hỏi #421

What is Amazon CloudWatch?

## Options

**A.** A code repository with customizable build and team commit features.
**B.** A metrics repository with customizable notification thresholds and channels.
**C.** A security configuration repository with threat analytics.
**D.** A rule repository of a web application firewall with automated vulnerability prevention features.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Câu hỏi định nghĩa `Amazon CloudWatch`.
- **Existing Resources:** Không có — bài toán khái niệm.
- **Current Issue/Goal:** Nhận diện mô tả đúng về CloudWatch.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `Amazon CloudWatch` | Dịch vụ monitoring metrics, logs, events, alarms |
| `metrics repository` | Lưu trữ metrics (CPU, network, disk…) từ AWS resources |
| `notification thresholds` | CloudWatch Alarms kích hoạt SNS khi vượt ngưỡng |
| `code repository` | Mô tả `AWS CodeCommit` |
| `security configuration` | Mô tả `AWS Security Hub` / `Inspector` |
| `web application firewall` | Mô tả `AWS WAF` |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Định nghĩa dịch vụ
- **Constraints:** Single-answer

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: B**

**Giải thích:**
- `Amazon CloudWatch` cung cấp khả năng thu thập và lưu trữ **metrics** từ AWS resources (EC2, RDS, ELB, Lambda…), logs, events.
- CloudWatch Alarms cho phép đặt ngưỡng (threshold) và gửi thông báo qua `Amazon SNS` (email, SMS, HTTPS endpoint) hoặc trigger Auto Scaling.
- Còn hỗ trợ `CloudWatch Dashboards` để visualize metrics, `Logs Insights` để query logs.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- "Code repository with build features" mô tả `AWS CodeCommit` (Git) kết hợp `CodeBuild`.
- CloudWatch không quản lý source code.

**❌ Đáp án C:**
- "Security configuration repository with threat analytics" mô tả `AWS Security Hub` (tổng hợp security alerts) hoặc `Amazon GuardDuty` (threat detection).
- CloudWatch tập trung vào metrics/logs, không phải threat analytics.

**❌ Đáp án D:**
- "Rule repository of WAF with vulnerability prevention" mô tả `AWS WAF` (Web Application Firewall) với managed rule groups.
- CloudWatch không phải WAF.

## 6. MẺO GHI NHỚ (Memory Hook)
🧠 *`CloudWatch` = metrics + logs + alarms. `CodeCommit` = code. `WAF` = web firewall. `GuardDuty` = threat analytics.*
