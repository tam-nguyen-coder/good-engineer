# Câu hỏi #88

Which of the following services allows you to run containerized applications on a cluster of EC2 instances?

## Options

**A.** Amazon ECS.
**B.** AWS Data Pipeline.
**C.** AWS Cloud9.
**D.** AWS Personal Health Dashboard.

- **Existing Resources:** Đề nhắc tới: `EC2` — dịch vụ compute (máy chủ ảo) trên AWS Cloud.
- **Current Issue/Goal:** Xác định đúng khái niệm/dịch vụ/tính năng AWS được hỏi tới.

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: A**

**Giải thích:**
