# Câu hỏi #431

Which AWS service or feature can enhance network security by blocking requests from a particular network for a web application on AWS? (Select TWO)

## Options

**A.** AWS WAF.
**B.** AWS Trusted Advisor.
**C.** AWS Direct Connect.
**D.** AWS Organizations.
**E.** Network ACLs.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Câu hỏi multi-answer về dịch vụ/feature giúp chặn requests từ network cụ thể cho web application.
- **Existing Resources:** Một web application trên AWS.
- **Current Issue/Goal:** Tăng cường bảo mật mạng, chặn request từ IP/network xác định.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `block requests from a particular network` | Chặn traffic từ IP/IP range |
| `AWS WAF` | Web Application Firewall — chặn theo IP, country, signature, SQLi, XSS |
| `Network ACLs` | Stateless firewall ở subnet — chặn theo IP, port, protocol |
| `AWS Trusted Advisor` | Best practice check, không block traffic |
| `AWS Direct Connect` | Private network, không block web traffic |
| `AWS Organizations` | Quản lý multi-account, không block traffic |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Network security (multi-answer)
- **Constraints:** Phải chặn được request từ network cụ thể cho web app

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: A và E**

**Giải thích:**
- `AWS WAF` (Web Application Firewall) cho phép tạo rule chặn request theo IP address, country, header, query string, body. Kết hợp với `CloudFront`/`ALB`/`API Gateway` để bảo vệ web application khỏi SQLi, XSS, bots, DDoS.
- `Network ACLs` (NACL) hoạt động ở subnet level, cho phép `ALLOW` hoặc `DENY` rule stateless theo source/destination IP, port range, protocol. Có thể dùng để chặn IP/network cụ thể truy cập vào subnet.
- Cả hai đều là lớp bảo mật mạng quan trọng; WAF hoạt động ở layer 7, NACL ở layer 3/4.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án B:**
- `AWS Trusted Advisor` cung cấp check best practice, có thể cảnh báo SG mở nhưng **không chủ động block traffic**.
- Đáp án này phù hợp cho advisory, không phải enforcement.

**❌ Đáp án C:**
- `AWS Direct Connect` là private network, không phải công cụ chặn traffic.
- Đáp án này phù hợp cho hybrid connectivity.

**❌ Đáp án D:**
- `AWS Organizations` quản lý nhiều account và SCP permission guardrails, không block network traffic.
- Đáp án này phù hợp cho multi-account governance.

## 6. MẺO GHI NHỚ (Memory Hook)
🧠 *Block traffic theo IP: `WAF` (layer 7) + `NACL` (layer 3/4) + `Security Group` (stateful, instance-level). Trust Advisor = chỉ cảnh báo.*
