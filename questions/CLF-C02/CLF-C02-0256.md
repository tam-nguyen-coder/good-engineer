# Câu hỏi #256

A customer is planning to migrate their Microsoft SQL Server databases to AWS. Which AWS Services can the customer use to run their Microsoft SQL Server database on AWS? (Choose TWO)

## Options

**A.** AWS Fargate.
**B.** Amazon Elastic Compute Cloud.
**C.** Amazon RDS.
**D.** AWS Database Migration service (DMS).
**E.** AWS Lambda.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Ngữ cảnh đề bài: _A customer is planning to migrate their Microsoft SQL Server databases to AWS. Which AWS Services can the customer use t..._
- **Existing Resources:** Không có tài nguyên cụ thể — câu hỏi thuộc dạng lý thuyết/khái niệm.
- **Current Issue/Goal:** Hệ thống cần co giãn linh hoạt theo tải, không bị giới hạn bởi cấu hình cố định.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| ``EC2`` | dịch vụ compute (máy chủ ảo) trên AWS Cloud |
| ``RDS`` | dịch vụ database quan hệ được quản lý |
| ``Lambda`` | dịch vụ serverless compute chạy theo sự kiện |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Scalability / Elasticity
- **Constraints:** Khả năng mở rộng theo tải

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: B và C**

**Giải thích:**
Đề yêu cầu chọn nhiều đáp án. Hai phương án **B** và **C** cùng đáp ứng các tiêu chí của câu hỏi.
- **B** — `EC2`: dịch vụ compute (máy chủ ảo) trên AWS Cloud.
- **C** — `RDS`: dịch vụ database quan hệ được quản lý.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- Tại sao sai: `Fargate` (serverless compute cho container) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `Fargate`.

**❌ Đáp án D:**
- Tại sao sai: Đáp án này không trực tiếp giải quyết yêu cầu cụ thể của đề, hoặc dịch vụ/tính năng này có giới hạn so với đáp án đúng.
- Khi nào đúng: Đáp án này sẽ đúng trong bối cảnh khác, nơi ưu tiên khác với yêu cầu của câu hỏi này.

**❌ Đáp án E:**
- Tại sao sai: `Lambda` (dịch vụ serverless compute chạy theo sự kiện) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `Lambda`.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **B và C** = `EC2`. Scale = Auto Scaling + Load Balancer, thêm/bớt instance theo tải.*
