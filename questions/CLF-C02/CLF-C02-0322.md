# Câu hỏi #322

Which of the following services allows you to install and run custom relational database software?

## Options

**A.** Amazon EC2.
**B.** Amazon Cognito.
**C.** Amazon RDS.
**D.** Amazon Inspector.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Đề hỏi về khái niệm/dịch vụ AWS liên quan tới: _Which of the following services allows you to install and run custom relational ..._
- **Existing Resources:** Không có tài nguyên cụ thể — câu hỏi thuộc dạng lý thuyết/khái niệm.
- **Current Issue/Goal:** Xác định đúng khái niệm/dịch vụ/tính năng AWS được hỏi tới.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| ``EC2`` | dịch vụ compute (máy chủ ảo) trên AWS Cloud |
| ``RDS`` | dịch vụ database quan hệ được quản lý |
| ``Inspector`` | dịch vụ quét lỗ hổng bảo mật tự động |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Database
- **Constraints:** Tuân theo best practice AWS

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: A**

**Giải thích:**
Đáp án `EC2` đúng vì dịch vụ compute (máy chủ ảo) trên AWS Cloud, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `EC2` đóng vai trò: chạy ứng dụng, web server, batch processing.

**❌ Đáp án B:**
- Tại sao sai: `Cognito` (dịch vụ identity cho user) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `Cognito`.

**❌ Đáp án C:**
- Tại sao sai: `RDS` (dịch vụ database quan hệ được quản lý) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `RDS`.

**❌ Đáp án D:**
- Tại sao sai: `Inspector` (dịch vụ quét lỗ hổng bảo mật tự động) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `Inspector`.
