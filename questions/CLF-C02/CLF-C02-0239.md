# Câu hỏi #239

When granting permissions to applications running on Amazon EC2 instances, which of the following is considered best practice?

## Options

**A.** Generate new IAM access keys every time you delegate permissions.
**B.** Store the required AWS credentials directly within the application code.
**C.** Use temporary security credentials (IAM roles) instead of long-term access keys.
**D.** Do nothing; Applications that run on Amazon EC2 instances do not need permission to interact with other AWS services or resources.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Ngữ cảnh đề bài: _When granting permissions to applications running on Amazon EC2 instances, which of the following is considered best pra..._
- **Existing Resources:** Đề nhắc tới: `EC2` — dịch vụ compute (máy chủ ảo) trên AWS Cloud; `IAM` — Identity and Access Management - quản lý quyền truy cập.
- **Current Issue/Goal:** Lựa chọn giải pháp bảo mật tốt nhất theo best practice của AWS.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `Security` | Yêu cầu bảo mật cao |
| ``EC2`` | dịch vụ compute (máy chủ ảo) trên AWS Cloud |
| ``IAM`` | Identity and Access Management - quản lý quyền truy cập |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Most secure / Security best practice
- **Constraints:** Bảo mật dữ liệu và quyền truy cập

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: C**

**Giải thích:**
Đáp án `IAM` đúng vì Identity and Access Management - quản lý quyền truy cập, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `IAM` đóng vai trò: quản lý user, group, role, policy cho AWS resources.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- Tại sao sai: `IAM` (Identity and Access Management - quản lý quyền truy cập) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu bảo mật chính là quản lý user, group, role, policy cho AWS resources này.

**❌ Đáp án B:**
- Tại sao sai: Đáp án này không trực tiếp giải quyết yêu cầu cụ thể của đề, hoặc dịch vụ/tính năng này có giới hạn so với đáp án đúng.
- Khi nào đúng: Đáp án này sẽ đúng trong bối cảnh khác, nơi ưu tiên khác với yêu cầu của câu hỏi này.

**❌ Đáp án D:**
- Tại sao sai: `EC2` (dịch vụ compute (máy chủ ảo) trên AWS Cloud) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu bảo mật chính là chạy ứng dụng, web server, batch processing này.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **C** = `EC2`. Bảo mật = defense-in-depth: nhiều lớp, nguyên tắc least privilege.*
