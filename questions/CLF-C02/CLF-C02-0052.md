# Câu hỏi #52

Which service provides object-level storage in AWS?

## Options

**A.** Amazon EBS.
**B.** Amazon Instance Store.
**C.** Amazon EFS.
**D.** Amazon S3.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Đề hỏi về khái niệm/dịch vụ AWS liên quan tới: _Which service provides object-level storage in AWS?_
- **Existing Resources:** Không có tài nguyên cụ thể — câu hỏi thuộc dạng lý thuyết/khái niệm.
- **Current Issue/Goal:** Lựa chọn đáp án đúng nhất theo yêu cầu của đề.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| ``S3`` | dịch vụ lưu trữ object bền vững (11x 9s) |
| ``EBS`` | Elastic Block Store - ổ đĩa block cho EC2 |
| ``EFS`` | Elastic File System - file system NFS managed |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Storage
- **Constraints:** Tuân theo best practice AWS

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: D**

**Giải thích:**
Đáp án `S3` đúng vì dịch vụ lưu trữ object bền vững (11x 9s), phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `S3` đóng vai trò: lưu trữ file, backup, static website, data lake, log archive.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- Tại sao sai: `EBS` (Elastic Block Store - ổ đĩa block cho EC2) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `EBS`.

**❌ Đáp án B:**
- Tại sao sai: Đáp án này không trực tiếp giải quyết yêu cầu cụ thể của đề, hoặc dịch vụ/tính năng này có giới hạn so với đáp án đúng.
- Khi nào đúng: Đáp án này sẽ đúng trong bối cảnh khác, nơi ưu tiên khác với yêu cầu của câu hỏi này.

**❌ Đáp án C:**
- Tại sao sai: `EFS` (Elastic File System - file system NFS managed) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `EFS`.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **D** = `S3` — phù hợp với yêu cầu đề bài.*
