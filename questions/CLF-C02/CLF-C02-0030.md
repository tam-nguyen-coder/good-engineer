# Câu hỏi #30

Which of the following can be described as a global content delivery network (CDN) service?

## Options

**A.** AWS VPN.
**B.** AWS Direct Connect.
**C.** AWS Regions.
**D.** Amazon CloudFront.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Đề hỏi về dịch vụ `CDN` (Content Delivery Network) toàn cầu của AWS.
- **Existing Resources:** Không có tài nguyên cụ thể.
- **Current Issue/Goal:** Xác định CDN service của AWS.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `global content delivery network (CDN)` | Mạng phân phối nội dung tới user toàn cầu |
| `Amazon CloudFront` | CDN chính thức của AWS, dùng `Edge Locations` |
| `AWS VPN` | Kết nối mạng riêng ảo vào AWS, không phải CDN |
| `AWS Direct Connect` | Đường truyền riêng vật lý on-premises ↔ AWS, không phải CDN |
| `AWS Regions` | Vùng địa lý, không phải CDN |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Khái niệm — AWS CDN service
- **Constraints:** Phải là CDN global

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: D**

**Giải thích:**
- `Amazon CloudFront` là dịch vụ CDN (Content Delivery Network) toàn cầu của AWS, phân phối nội dung (web, video, API, download) thông qua hàng trăm `Edge Locations` trên thế giới.
- Tự động cache nội dung gần user nhất → giảm latency, tăng tốc độ tải.
- Hỗ trợ tích hợp với `S3`, `EC2`, `ALB`, `API Gateway`, `MediaStore`, ... làm origin. Có thể kết hợp `Lambda@Edge` để chạy code tại edge.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- `AWS VPN` (Virtual Private Network) tạo kết nối mã hóa từ on-premises hoặc user vào `VPC` của AWS qua Internet. Không phải CDN.

**❌ Đáp án B:**
- `AWS Direct Connect` cung cấp kết nối mạng riêng vật lý (dedicated fiber) giữa data center on-premises và AWS, giảm chi phí bandwidth và tăng stability. Không phải CDN.

**❌ Đáp án C:**
- `AWS Regions` là vùng địa lý chứa nhiều data center, là cấp tổ chức cao nhất trong AWS global infrastructure, không phải CDN.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *"CDN toàn cầu của AWS = `CloudFront`. `VPN` = mạng riêng ảo, `Direct Connect` = đường truyền riêng, `Region` = vùng địa lý."*
