# Câu hỏi #36

Which of the following must an IAM user provide to interact with AWS services using the AWS Command Line Interface (AWS CLI)?

## Options

**A.** Access keys.
**B.** Secret token.
**C.** UserID.
**D.** User name and password.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Đề hỏi về thông tin xác thực cần thiết để một `IAM user` tương tác với AWS thông qua `AWS CLI`.
- **Existing Resources:** IAM user account đã tạo trong AWS.
- **Current Issue/Goal:** Xác định credential phù hợp cho programmatic access qua CLI.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `AWS CLI` | Command line interface, giao tiếp với AWS qua API call |
| `Access keys` | Gồm `Access Key ID` + `Secret Access Key`, dùng cho CLI/SDK |
| `User name + password` | Dùng cho `Management Console` qua web, không dùng cho CLI |
| `Secret token` | `Session Token` chỉ cần khi dùng temporary credentials từ STS |
| `UserID` | Không phải credential dùng cho AWS CLI |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Khái niệm — IAM credentials cho programmatic access
- **Constraints:** Phải là credential dùng cho CLI (không phải console)

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: A**

**Giải thích:**
- Để tương tác với AWS thông qua `AWS CLI` (hoặc SDK), cần sử dụng `Access Keys` gồm `Access Key ID` và `Secret Access Key`.
- Access key được tạo cho `IAM user` trong console hoặc qua API, cấu hình bằng lệnh `aws configure`.
- Mỗi IAM user có tối đa 2 access key, nên xoay key định kỳ để tăng bảo mật.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án B:**
- `Secret token` (Session Token) chỉ cần khi dùng temporary credentials từ `AWS STS` (ví dụ khi `AssumeRole` hoặc `SSO`). Với access key thường, không cần session token riêng.

**❌ Đáp án C:**
- `UserID` (ví dụ `AIDA...`) chỉ là định danh nội bộ của IAM user, không phải credential để CLI xác thực.

**❌ Đáp án D:**
- `User name + password` chỉ dùng cho `AWS Management Console` (web GUI), không dùng cho CLI.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *"CLI/SDK = `Access Key ID` + `Secret Access Key`. Console = username + password. STS/SSO = thêm `Session Token`."*
