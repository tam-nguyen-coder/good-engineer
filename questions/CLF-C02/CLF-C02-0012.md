# Câu hỏi #12

A company has decided to migrate its Oracle database to AWS. Which AWS service can help achieve this without negatively impacting the functionality of the source database?

## Options

**A.** AWS OpsWorks.
**B.** AWS Database Migration Service.
**C.** AWS Server Migration Service.
**D.** AWS Application Discovery Service.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Công ty muốn migrate `Oracle database` lên AWS mà **không ảnh hưởng** đến database nguồn (homogeneous/heterogeneous, near-zero downtime).
- **Existing Resources:** `Oracle database` đang chạy on-premises.
- **Current Issue/Goal:** Chọn dịch vụ AWS phù hợp để migrate database với minimal disruption.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `migrate Oracle database` | Cần công cụ migrate database cụ thể |
| `without negatively impacting` | Cần near-zero downtime, source vẫn hoạt động |
| `AWS Database Migration Service (DMS)` | Service chuyên migrate database (homogeneous & heterogeneous) |
| `AWS Server Migration Service` | Migrate `EC2`/on-prem VM, không phải database |
| `AWS Application Discovery Service` | Chỉ discover inventory, không migrate |
| `AWS OpsWorks` | Configuration management (Chef/Puppet) |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Chọn dịch vụ migration đúng
- **Constraints:** Phải là database migration, không ảnh hưởng source

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: B**

**Giải thích:**
- `AWS Database Migration Service (DMS)` là dịch vụ chuyên migrate database (Oracle, SQL Server, MySQL, PostgreSQL, ...) sang AWS, hỗ trợ cả homogeneous (cùng loại DB) và heterogeneous (khác loại DB, ví dụ Oracle → Aurora PostgreSQL).
- DMS cho phép source database tiếp tục hoạt động trong suốt quá trình migration, đảm bảo near-zero downtime.
- Kết hợp với `AWS Schema Conversion Tool (SCT)` để convert schema khi migrate sang engine khác.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- `AWS OpsWorks` là configuration management dùng Chef/Puppet để quản lý server, không phải công cụ migrate database.

**❌ Đáp án C:**
- `AWS Server Migration Service (SMS)` dùng để migrate on-premises VM vào `EC2`, không phải database engine.

**❌ Đáp án D:**
- `AWS Application Discovery Service` chỉ giúp khảo sát và lập inventory hệ thống on-premises, không thực hiện migration.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *"Migrate database = `DMS` (Database Migration Service). Migrate server/VM = `SMS`. Khảo sát = `Application Discovery`. Đừng lẫn!"*
