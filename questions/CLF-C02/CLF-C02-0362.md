# Câu hỏi #362

What is the recommended storage option when hosting an often-changing database on an Amazon EC2 instance?

## Options

**A.** Amazon EBS.
**B.** Amazon RDS.
**C.** You can't run a database inside an Amazon EC2 instance.
**D.** Amazon DynamoDB.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Ngữ cảnh đề bài: _What is the recommended storage option when hosting an often-changing database on an Amazon EC2 instance?_
- **Existing Resources:** Đề nhắc tới: `EC2` — dịch vụ compute (máy chủ ảo) trên AWS Cloud.
- **Current Issue/Goal:** Xác định đúng khái niệm/dịch vụ/tính năng AWS được hỏi tới.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| ``EC2`` | dịch vụ compute (máy chủ ảo) trên AWS Cloud |
| ``RDS`` | dịch vụ database quan hệ được quản lý |
| ``DynamoDB`` | dịch vụ NoSQL key-value được quản lý hoàn toàn |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Storage
- **Constraints:** Tuân theo best practice AWS

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: A**

**Giải thích:**
Đáp án `EBS` đúng vì Elastic Block Store - ổ đĩa block cho EC2, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `EBS` đóng vai trò: lưu trữ persistent cho EC2, snapshot, multi-attach.

- Tại sao sai: `RDS` (dịch vụ database quan hệ được quản lý) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `RDS`.

- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `EC2`.

**❌ Đáp án D:**
- Tại sao sai: `DynamoDB` (dịch vụ NoSQL key-value được quản lý hoàn toàn) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `DynamoDB`.
