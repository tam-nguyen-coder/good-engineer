# Câu hỏi #543

How can a user protect against AWS service disruptions if a natural disaster affects an entire geographic area?

## Options

**A.** Deploy applications across multiple Availability Zones within an AWS Region.
**B.** Use a hybrid cloud computing deployment model within the geographic area.
**C.** Deploy applications across multiple AWS Regions.
**D.** Store application artifacts using AWS Artifact and replicate them across multiple AWS Regions.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Ngữ cảnh đề bài: _How can a user protect against AWS service disruptions if a natural disaster affects an entire geographic area?_
- **Existing Resources:** Không có tài nguyên cụ thể — câu hỏi thuộc dạng lý thuyết/khái niệm.
- **Current Issue/Goal:** Đảm bảo hệ thống có tính sẵn sàng cao, chịu lỗi, khôi phục nhanh khi sự cố.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `Security` | Yêu cầu bảo mật cao |
| `Hybrid / On-premises` | Kết nối môi trường on-premises với AWS |
| ``AWS Artifact`` | cổng truy cập tài liệu compliance |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Highly available / Fault-tolerant
- **Constraints:** Yêu cầu tính sẵn sàng cao, không gián đoạn

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: C**

**Giải thích:**
Đáp án **C** mô tả đúng khái niệm/dịch vụ mà đề đang hỏi tới.
Đây là đáp án phù hợp nhất với best practice của AWS trong tình huống này.

**❌ Đáp án A:**
- Tại sao sai: Đáp án này không trực tiếp giải quyết yêu cầu cụ thể của đề, hoặc dịch vụ/tính năng này có giới hạn so với đáp án đúng.

- Tại sao sai: Đáp án này không trực tiếp giải quyết yêu cầu cụ thể của đề, hoặc dịch vụ/tính năng này có giới hạn so với đáp án đúng.
- Khi nào đúng: Đáp án này sẽ đúng trong bối cảnh khác, nơi ưu tiên khác với yêu cầu của câu hỏi này.
