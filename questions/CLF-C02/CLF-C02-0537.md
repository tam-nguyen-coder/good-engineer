# Câu hỏi #537

Which of the following AWS services can be used to serve large amounts of online video content with the lowest possible latency? (Select TWO)

## Options

**A.** appGateway.
**B.** Amazon S3.
**C.** Amazon Elastic File System (EFS).
**D.** Amazon Glacier.
**E.** Amazom CloudFront.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Ngữ cảnh đề bài: _Which of the following AWS services can be used to serve large amounts of online video content with the lowest possible ..._
- **Existing Resources:** Không có tài nguyên cụ thể — câu hỏi thuộc dạng lý thuyết/khái niệm.
- **Current Issue/Goal:** Lựa chọn giải pháp có hiệu năng cao nhất, độ trễ thấp nhất.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| ``S3`` | dịch vụ lưu trữ object bền vững (11x 9s) |
| ``CloudFront`` | CDN (Content Delivery Network) toàn cầu |
| ``Glacier`` | dịch vụ lưu trữ lạnh giá rẻ |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Best performance / Lowest latency
- **Constraints:** Hiệu năng và độ trễ thấp

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: B và E**

**Giải thích:**
Đề yêu cầu chọn nhiều đáp án. Hai phương án **B** và **E** cùng đáp ứng các tiêu chí của câu hỏi.
- **B** — `S3`: dịch vụ lưu trữ object bền vững (11x 9s).
- **E** — `CloudFront`: CDN (Content Delivery Network) toàn cầu.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- Tại sao sai: Đáp án này không trực tiếp giải quyết yêu cầu cụ thể của đề, hoặc dịch vụ/tính năng này có giới hạn so với đáp án đúng.
- Khi nào đúng: Đáp án này sẽ đúng trong bối cảnh khác, nơi ưu tiên khác với yêu cầu của câu hỏi này.

**❌ Đáp án C:**
- Tại sao sai: `EFS` (Elastic File System - file system NFS managed) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu chia sẻ file giữa nhiều EC2, Linux, scale tự động được ưu tiên hơn hiệu năng.

**❌ Đáp án D:**
- Tại sao sai: `Glacier` (dịch vụ lưu trữ lạnh giá rẻ) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu lưu trữ dài hạn, archive, backup ít truy cập được ưu tiên hơn hiệu năng.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **B và E** = `S3`. Hiệu năng = edge location + cache + chọn instance/region gần user.*
