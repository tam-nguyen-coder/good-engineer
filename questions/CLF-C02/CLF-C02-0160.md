# Câu hỏi #160

A company needs to track resource changes using the API call history. Which AWS service can help the company achieve this goal?

## Options

**A.** AWS Config.
**B.** Amazon CloudWatch.
**C.** AWS CloudTrail.
**D.** AWS CloudFormation.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Tình huống được mô tả: _A company needs to track resource changes using the API call history. Which AWS service can help the company achieve thi..._
- **Existing Resources:** Đề nhắc tới: `CloudTrail` — dịch vụ audit API call.
- **Current Issue/Goal:** Cần giám sát, theo dõi và ghi nhận hoạt động của hệ thống/tài khoản.

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: C**

**Giải thích:**
Đáp án `CloudTrail` đúng vì dịch vụ audit API call, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `CloudTrail` đóng vai trò: ghi lại mọi API call vào tài khoản AWS, hỗ trợ kiểm tra bảo mật.
