# Câu hỏi #418

Which AWS feature will reduce the customer's total cost of ownership (TCO)?

## Options

**A.** Shared responsibility security model.
**B.** Single tenancy.
**C.** Elastic computing.
**D.** Encryption.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Câu hỏi về đặc tính AWS giúp giảm Total Cost of Ownership (TCO) cho khách hàng.
- **Existing Resources:** Không có cụ thể — bài toán khái niệm.
- **Current Issue/Goal:** Xác định feature/mô hình giúp tiết kiệm tổng chi phí sở hữu.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `reduce TCO` | Tổng chi phí sở hữu thấp hơn on-premises |
| `Elastic computing` | Scale theo demand, không over-provision |
| `Shared responsibility` | Về security, không trực tiếp giảm TCO |
| `Single tenancy` | Dedicated host/instance, chi phí cao hơn shared |
| `Encryption` | Về bảo mật, không phải cost optimization |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Khái niệm / Cost optimization
- **Constraints:** Câu hỏi single-answer

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: C**

**Giải thích:**
- `Elastic computing` (điện toán đàn hồi) cho phép doanh nghiệp scale resource up/down theo nhu cầu thực tế, chỉ trả cho những gì dùng (pay-as-you-go).
- Thay vì mua server dư thừa cho peak load (TCO cao), khách hàng dùng `Auto Scaling`, `Spot Instances`, `Lambda`… để tối ưu chi phí.
- Đây là một trong những lợi ích kinh tế cốt lõi của cloud computing, giúp giảm đáng kể TCO so với on-premises.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- `Shared responsibility security model` giúp phân chia rõ trách nhiệm bảo mật, không trực tiếp giảm TCO (vẫn phải trả chi phí security).
- Đáp án này phù hợp cho câu hỏi về security model.

**❌ Đáp án B:**
- `Single tenancy` (EC2 Dedicated Hosts/Instances) thường **tăng** chi phí so với shared tenancy.
- Đáp án này chỉ cần thiết khi có yêu cầu compliance/licensing đặc biệt.

**❌ Đáp án D:**
- `Encryption` có thể phát sinh thêm chi phí (KMS key, performance overhead), là về bảo mật chứ không phải cost optimization.
- Đáp án này thuộc Security pillar.

## 6. MẺO GHI NHỚ (Memory Hook)
🧠 *Giảm TCO = `elasticity` (pay-as-you-go) + `Auto Scaling` + `Spot`. Tránh `single tenancy` và `over-provision`.*
