# Câu hỏi #406

Which AWS service allows companies to connect an Amazon VPC to an on-premises data center?

## Options

**A.** AWS VPN.
**B.** Amazon Redshift.
**C.** API Gateway.
**D.** Amazon Direct Connect.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Câu hỏi về dịch vụ kết nối hybrid giữa `Amazon VPC` và data center on-premises.
- **Existing Resources:** Có VPC trên AWS và một data center tại chỗ.
- **Current Issue/Goal:** Tìm dịch vụ giúp kết nối an toàn giữa hai môi trường.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `connect an Amazon VPC to an on-premises data center` | Hybrid connectivity |
| `AWS VPN` | Kết nối mã hóa qua internet công cộng (IPsec VPN) |
| `AWS Direct Connect` | Đường truyền riêng vật lý (private fiber) từ DC đến AWS |
| `Amazon Redshift` | Data warehouse, không liên quan đến kết nối mạng |
| `API Gateway` | Front-door cho REST/WebSocket API, không phải hybrid networking |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Xác định dịch vụ kết nối hybrid
- **Constraints:** Kết nối VPC ↔ on-prem datacenter

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: A**

**Giải thích:**
- `AWS VPN` (gồm `Site-to-Site VPN`) cho phép thiết lập đường hầm mã hóa `IPsec` giữa VPC và cơ sở on-premises, dẫn qua internet công cộng.
- Đây là phương án phổ biến nhất cho hybrid cloud vì triển khai nhanh, chi phí thấp, mã hóa toàn trình (encryption in transit).
- Lưu ý: `AWS Direct Connect` cũng kết nối được nhưng cần cáp quang riêng và thời gian thiết lập lâu hơn.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án B:**
- `Amazon Redshift` là dịch vụ data warehouse quan hệ (columnar), không dùng để kết nối mạng.
- Đáp án này chỉ dùng cho phân tích dữ liệu lớn (OLAP).

**❌ Đáp án C:**
- `Amazon API Gateway` là dịch vụ tạo, publish, maintain, monitor, và bảo vệ REST/HTTP/WebSocket API; không phải công cụ kết nối hybrid network.
- Đáp án này phù hợp cho backend API cho web/mobile app.

**❌ Đáp án D:**
- `AWS Direct Connect` cũng kết nối VPC với on-prem nhưng là đường truyền vật lý riêng (private fiber), yêu cầu ký hợp đồng với partner và có thể mất vài tuần đến vài tháng để triển khai.
- Câu hỏi yêu cầu "service allows" mà không giới hạn điều kiện đặc biệt, nên `AWS VPN` được ưu tiên hơn nhờ tính phổ biến, sẵn có, triển khai nhanh.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Hybrid network nhanh + mã hóa = `Site-to-Site VPN` (qua internet). Cần băng thông ổn định + latency thấp = `Direct Connect`.*
