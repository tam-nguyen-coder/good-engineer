# Câu hỏi #423

Under the shared responsibility model, which of the following is a shared control between a customer and AWS?

## Options

**A.** Physical controls.
**B.** Patch management.
**C.** Zone security.
**D.** Data center auditing.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Câu hỏi xác định control thuộc **shared** (vừa AWS vừa khách hàng cùng chịu trách nhiệm ở những phần khác nhau) trong Shared Responsibility Model.
- **Existing Resources:** Không có — bài toán khái niệm.
- **Current Issue/Goal:** Phân biệt sole-AWS vs sole-customer vs shared.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `shared control` | Cả AWS và khách hàng cùng chịu phần việc của mình |
| `Patch management` | AWS patch hạ tầng/managed services; khách hàng patch EC2 OS, app |
| `Physical controls` | Sole-AWS |
| `Zone security` | Sole-AWS |
| `Data center auditing` | Sole-AWS |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Khái niệm Shared Responsibility
- **Constraints:** Single-answer

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: B**

**Giải thích:**
- `Patch management` là điển hình của **shared control**:
  - AWS chịu: patch hypervisor, host firmware, các managed services (RDS OS, Lambda runtime…).
  - Khách hàng chịu: patch OS trên EC2, application dependencies, container images, Lambda custom runtime.
- Các shared controls khác: configuration management, awareness & training.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- `Physical controls` (kiểm soát vật lý: camera, khóa, fence, guard) thuộc sole-AWS — khách hàng không thể/ không cần can thiệp.
- Đáp án này thuộc Security **of** the Cloud.

**❌ Đáp án C:**
- `Zone security` (bảo mật khu vực) là phần mở rộng của physical controls, cũng thuộc sole-AWS.
- Đáp án này thuộc Security **of** the Cloud.

**❌ Đáp án D:**
- `Data center auditing` (audit compliance cho datacenter) do AWS thực hiện (cung cấp qua AWS Artifact).
- Đáp án này thuộc Security **of** the Cloud.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Shared controls: `Patch management` + `Configuration management` + `Awareness & training`. Sole AWS: physical, environmental, zone, audit.*
