# Câu hỏi #360

Which AWS service provides the EASIEST way to set up and manage a secure, well-architected, multi-account AWS environment?

## Options

**A.** AWS Control Tower.
**B.** Amazon Macie.
**C.** AWS Systems Manager Patch Manager.
**D.** AWS Systems Manager Patch Manager AWS Security Hub.

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: A**

**Giải thích:**
Đáp án `Control Tower` đúng vì thiết lập và quản trị multi-account theo best practice, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `Control Tower` đóng vai trò: landing zone chuẩn hóa, guardrails tự động.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án B:**
- Tại sao sai: `Macie` (phát hiện dữ liệu nhạy cảm trong S3) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu bảo mật chính là quét S3 tìm PII, dữ liệu cá nhân, tài chính này.

**❌ Đáp án C:**
- Tại sao sai: `Systems Manager` (quản lý fleet EC2 và on-premises) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu bảo mật chính là patch, automation, parameter store, session manager này.

**❌ Đáp án D:**
- Tại sao sai: `Systems Manager` (quản lý fleet EC2 và on-premises) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu bảo mật chính là patch, automation, parameter store, session manager này.
