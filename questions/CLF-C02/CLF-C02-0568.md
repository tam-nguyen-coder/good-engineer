# Câu hỏi #568

A company wants to migrate its applications to a VPC on AWS. These applications will need to access on-premises resources. What combination of actions will enable the company to accomplish this goals? (Select TWO)

## Options

**A.** Use the AWS Service Catalog to identify a list of on-premises resources that can be migrated.
**B.** Build a VPN connection between an on-premises device and a virtual private gateway in the new VPC.
**C.** Use Amazon Athena to query data from the on-premises database servers.
**D.** Connect the company's on-premises data center to AWS using AWS Direct Connect.
**E.** Leverage Amazon CloudFront to restrict access to static web content provided through the company's on-premises web servers.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Tình huống được mô tả: _A company wants to migrate its applications to a VPC on AWS. These applications will need to access on-premises resource..._
- **Existing Resources:** Đề nhắc tới: `VPC` — Virtual Private Cloud - mạng riêng ảo trong AWS.
- **Current Issue/Goal:** Lựa chọn giải pháp bảo mật tốt nhất theo best practice của AWS.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `Hybrid / On-premises` | Kết nối môi trường on-premises với AWS |
| `IoT` | Thu thập dữ liệu từ thiết bị |
| `Web application` | Ứng dụng web/API |
| ``VPC`` | Virtual Private Cloud - mạng riêng ảo trong AWS |
| ``CloudFront`` | CDN (Content Delivery Network) toàn cầu |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Most secure / Security best practice
- **Constraints:** Bảo mật dữ liệu và quyền truy cập

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: B và D**

**Giải thích:**
Đề yêu cầu chọn nhiều đáp án. Hai phương án **B** và **D** cùng đáp ứng các tiêu chí của câu hỏi.
- **B** — `VPC`: Virtual Private Cloud - mạng riêng ảo trong AWS.
- **D** — `Direct Connect`: kết nối mạng riêng tốc độ cao tới AWS.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- Tại sao sai: Đáp án này không trực tiếp giải quyết yêu cầu cụ thể của đề, hoặc dịch vụ/tính năng này có giới hạn so với đáp án đúng.
- Khi nào đúng: Đáp án này sẽ đúng trong bối cảnh khác, nơi ưu tiên khác với yêu cầu của câu hỏi này.

**❌ Đáp án C:**
- Tại sao sai: `Athena` (query S3 bằng SQL) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu bảo mật chính là phân tích dữ liệu trong S3 mà không cần copy này.

**❌ Đáp án E:**
- Tại sao sai: `CloudFront` (CDN (Content Delivery Network) toàn cầu) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu bảo mật chính là phân phối nội dung tĩnh/động qua edge location, giảm latency này.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **B và D** = `VPC`. Bảo mật = defense-in-depth: nhiều lớp, nguyên tắc least privilege.*
