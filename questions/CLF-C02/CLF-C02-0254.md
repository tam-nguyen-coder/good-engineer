# Câu hỏi #254

Which AWS Service allows customers to create a template that programmatically defines policies and configurations of all AWS resources as code and so that the same template can be reused among multiple projects?

## Options

**A.** AWS CloudFormation.
**B.** AWS Config.
**C.** AWS CloudTrail.
**D.** AWS Auto Scaling.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Ngữ cảnh đề bài: _Which AWS Service allows customers to create a template that programmatically defines policies and configurations of all..._
- **Existing Resources:** Đề nhắc tới: `IAM` — Identity and Access Management - quản lý quyền truy cập.
- **Current Issue/Goal:** Lựa chọn giải pháp bảo mật tốt nhất theo best practice của AWS.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| ``IAM`` | Identity and Access Management - quản lý quyền truy cập |
| ``CloudTrail`` | dịch vụ audit API call |
| ``CloudFormation`` | Infrastructure as Code |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Most secure / Security best practice
- **Constraints:** Bảo mật dữ liệu và quyền truy cập

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: A**

**Giải thích:**
Đáp án `CloudFormation` đúng vì Infrastructure as Code, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `CloudFormation` đóng vai trò: định nghĩa hạ tầng bằng template JSON/YAML, tự động provision.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án B:**
- Tại sao sai: `AWS Config` (dịch vụ quản lý cấu hình tài nguyên) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu bảo mật chính là theo dõi thay đổi cấu hình, đánh giá compliance này.

**❌ Đáp án C:**
- Tại sao sai: `CloudTrail` (dịch vụ audit API call) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu bảo mật chính là ghi lại mọi API call vào tài khoản AWS, hỗ trợ kiểm tra bảo mật này.

**❌ Đáp án D:**
- Tại sao sai: `Auto Scaling` (tự động scale số lượng EC2) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu bảo mật chính là thêm/giảm instance theo demand, đảm bảo availability này.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **A** = `IAM`. Bảo mật = defense-in-depth: nhiều lớp, nguyên tắc least privilege.*
