# Câu hỏi #527

Which AWS service provides the ability to manage infrastructure as code?

## Options

**A.** AWS CodePipeline.
**B.** AWS CodeDeploy.
**C.** AWS Direct Connect.
**D.** AWS CloudFormation.


---

- **Scenario:** Đề hỏi về khái niệm/dịch vụ AWS liên quan tới: _Which AWS service provides the ability to manage infrastructure as code?_

- **Current Issue/Goal:** Lựa chọn đáp án đúng nhất theo yêu cầu của đề.

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: D**

**Giải thích:**
Đáp án `CloudFormation` đúng vì Infrastructure as Code, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `CloudFormation` đóng vai trò: định nghĩa hạ tầng bằng template JSON/YAML, tự động provision.
