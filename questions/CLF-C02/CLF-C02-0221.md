# Câu hỏi #221

A company is seeking to deploy an existing .NET application onto AWS as quickly as possible. Which AWS Service should the customer use to achieve this goal?

## Options

**A.** Amazon SNS.
**B.** AWS Elastic Beanstalk.
**C.** AWS Systems Manager.
**D.** AWS Trusted Advisor.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Tình huống được mô tả: _A company is seeking to deploy an existing .NET application onto AWS as quickly as possible. Which AWS Service should th..._
- **Existing Resources:** Không có tài nguyên cụ thể — câu hỏi thuộc dạng lý thuyết/khái niệm.
- **Current Issue/Goal:** Hệ thống cần co giãn linh hoạt theo tải, không bị giới hạn bởi cấu hình cố định.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| ``SNS`` | Simple Notification Service - pub/sub messaging |
| ``Elastic Beanstalk`` | PaaS - nền tảng triển khai ứng dụng |
| ``Trusted Advisor`` | cố vấn tối ưu chi phí, bảo mật, hiệu năng |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Scalability / Elasticity
- **Constraints:** Khả năng mở rộng theo tải

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: B**

**Giải thích:**
Đáp án `Elastic Beanstalk` đúng vì PaaS - nền tảng triển khai ứng dụng, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `Elastic Beanstalk` đóng vai trò: deploy code mà AWS tự lo capacity balancing, scaling.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- Tại sao sai: `SNS` (Simple Notification Service - pub/sub messaging) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `SNS`.

**❌ Đáp án C:**
- Tại sao sai: `Systems Manager` (quản lý fleet EC2 và on-premises) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `Systems Manager`.

**❌ Đáp án D:**
- Tại sao sai: `Trusted Advisor` (cố vấn tối ưu chi phí, bảo mật, hiệu năng) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `Trusted Advisor`.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **B** = `SNS`. Scale = Auto Scaling + Load Balancer, thêm/bớt instance theo tải.*
