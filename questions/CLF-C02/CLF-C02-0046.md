# Câu hỏi #46

Your company is developing a critical web application in AWS, and the security of the application is a top priority. Which of the following AWS services will provide infrastructure security optimization recommendations?

## Options

**A.** AWS Shield.
**B.** AWS Management Console.
**C.** AWS Secrets Manager.
**D.** AWS Trusted Advisor.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Công ty đang phát triển web application quan trọng trên AWS, **security là ưu tiên hàng đầu**. Cần service đưa ra **khuyến nghị tối ưu bảo mật hạ tầng**.
- **Existing Resources:** Web app + AWS infrastructure.
- **Current Issue/Goal:** Nhận khuyến nghị cụ thể để cải thiện security posture.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `security optimization recommendations` | Khuyến nghị cải thiện bảo mật, dạng check list |
| `AWS Trusted Advisor` | Đưa ra khuyến nghị về cost, security, performance, fault tolerance, service limits |
| `AWS Shield` | Chống DDoS, là dịch vụ bảo vệ chứ không phải tư vấn |
| `AWS Management Console` | Giao diện web, không phải advisory service |
| `AWS Secrets Manager` | Lưu trữ và xoay secret, không phải advisory |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Chọn service tư vấn bảo mật
- **Constraints:** Phải đưa ra security recommendations, không phải runtime protection

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: D**

**Giải thích:**
- `AWS Trusted Advisor` cung cấp khuyến nghị theo nhiều category, bao gồm **Security**: kiểm tra security group có rule quá mở, S3 bucket không có encryption, MFA chưa bật cho root account, IAM key chưa xoay, CloudTrail chưa được enable, ...
- Các check `Cost Optimization`, `Performance`, `Fault Tolerance`, `Service Limits` cũng rất hữu ích.
- Một số check cơ bản có sẵn cho mọi customer; toàn bộ check yêu cầu gói `Business`, `Enterprise On-Ramp`, hoặc `Enterprise Support`.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- `AWS Shield` là dịch vụ bảo vệ chống DDoS (Standard miễn phí, Advanced trả phí). Nó là lớp bảo vệ runtime, không phải công cụ đưa ra khuyến nghị tối ưu.

**❌ Đáp án B:**
- `AWS Management Console` chỉ là giao diện web, không có chức năng tư vấn bảo mật.

**❌ Đáp án C:**
- `AWS Secrets Manager` lưu trữ, mã hóa và tự động xoay secret (DB password, API key, ...). Đây là công cụ quản lý secret, không phải advisory.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *"Tư vấn tối ưu security/cost/performance = `Trusted Advisor`. `Shield` = chống DDoS runtime. `Secrets Manager` = quản lý secret."*
