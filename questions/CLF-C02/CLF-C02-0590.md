# Câu hỏi #590

Which AWS security service protects applications from distributed denial of service attacks with always-on detection and automatic inline mitigations?

## Options

**A.** Amazon Inspector.
**B.** AWS Web Application Firewall (AWS WAF).
**C.** Elastic Load Balancing (ELB).
**D.** AWS Shield.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Ngữ cảnh đề bài: _Which AWS security service protects applications from distributed denial of service attacks with always-on detection and..._
- **Existing Resources:** Không có tài nguyên cụ thể — câu hỏi thuộc dạng lý thuyết/khái niệm.
- **Current Issue/Goal:** Lựa chọn giải pháp bảo mật tốt nhất theo best practice của AWS.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `Security` | Yêu cầu bảo mật cao |
| `Web application` | Ứng dụng web/API |
| `DDoS protection` | Chống tấn công từ chối dịch vụ |
| ``Inspector`` | dịch vụ quét lỗ hổng bảo mật tự động |
| ``WAF`` | Web Application Firewall |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Most secure / Security best practice
- **Constraints:** Bảo mật dữ liệu và quyền truy cập

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: D**

**Giải thích:**
Đáp án `Shield` đúng vì dịch vụ chống DDoS, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `Shield` đóng vai trò: bảo vệ khỏi tấn công DDoS ở tầng network/application.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- Tại sao sai: `Inspector` (dịch vụ quét lỗ hổng bảo mật tự động) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu bảo mật chính là đánh giá security posture của EC2, container, Lambda này.

**❌ Đáp án B:**
- Tại sao sai: `WAF` (Web Application Firewall) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu bảo mật chính là lọc traffic web ngăn chặn SQL injection, XSS, bot này.

**❌ Đáp án C:**
- Tại sao sai: `Elastic Load Balancer` (cân bằng tải (ALB, NLB, GLB)) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu bảo mật chính là phân phối traffic tới nhiều target, health check này.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **D** = `Inspector`. Bảo mật = defense-in-depth: nhiều lớp, nguyên tắc least privilege.*
