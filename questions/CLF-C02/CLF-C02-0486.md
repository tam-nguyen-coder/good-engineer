# Câu hỏi #486

Which AWS service is a managed NoSQL database?

## Options

**A.** Amazon Redshift.
**B.** Amazon DynamoDB.
**C.** Amazon Aurora.
**D.** Amazon RDS for ManaDB.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Đề hỏi về khái niệm/dịch vụ AWS liên quan tới: _Which AWS service is a managed NoSQL database?_
- **Existing Resources:** Đề nhắc tới: `DynamoDB` — dịch vụ NoSQL key-value được quản lý hoàn toàn.
- **Current Issue/Goal:** Lựa chọn đáp án đúng nhất theo yêu cầu của đề.

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: B**

**Giải thích:**
Đáp án `DynamoDB` đúng vì dịch vụ NoSQL key-value được quản lý hoàn toàn, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `DynamoDB` đóng vai trò: lưu trữ dữ liệu phi quan hệ với độ trễ thấp ở mọi quy mô.
