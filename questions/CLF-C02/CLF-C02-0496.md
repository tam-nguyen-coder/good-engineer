# Câu hỏi #496

A company's web application currently has light dependencies on underlying components so when one component fails the entire web application fails. Applying which AWS Cloud design principle will address the current design issue?

## Options

**A.** Implementing elasticity enabling the application to scale up or scale down as demand changes.
**B.** Enabling several EC2 instances to run in parallel to achieve better performance.
**C.** Focusing on decoupling components by isolating them and ensuring individual components can function when other components.
**D.** Doubling EC2 computing resources to increase system fault tolerance.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Tình huống được mô tả: _A company's web application currently has light dependencies on underlying components so when one component fails the en..._
- **Existing Resources:** Không có tài nguyên cụ thể — câu hỏi thuộc dạng lý thuyết/khái niệm.
- **Current Issue/Goal:** Lựa chọn giải pháp có hiệu năng cao nhất, độ trễ thấp nhất.

|---------|-----------------|

- **Constraints:** Hiệu năng và độ trễ thấp

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: C**

**Giải thích:**
Đáp án `Compliance Programs` đúng vì chương trình tuân thủ (SOC, PCI, HIPAA, GDPR, ISO), phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `Compliance Programs` đóng vai trò: chứng nhận AWS đáp ứng tiêu chuẩn ngành.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- Tại sao sai: Đáp án này không trực tiếp giải quyết yêu cầu cụ thể của đề, hoặc dịch vụ/tính năng này có giới hạn so với đáp án đúng.
- Khi nào đúng: Đáp án này sẽ đúng trong bối cảnh khác, nơi ưu tiên khác với yêu cầu của câu hỏi này.

**❌ Đáp án B:**
- Tại sao sai: `EC2` (dịch vụ compute (máy chủ ảo) trên AWS Cloud) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu chạy ứng dụng, web server, batch processing được ưu tiên hơn hiệu năng.

**❌ Đáp án D:**
- Tại sao sai: `EC2` (dịch vụ compute (máy chủ ảo) trên AWS Cloud) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu chạy ứng dụng, web server, batch processing được ưu tiên hơn hiệu năng.
