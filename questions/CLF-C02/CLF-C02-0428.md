# Câu hỏi #428

Which of the following services will automatically scale with an expected increase in web traffic?

## Options

**A.** AWS CodePipeline.
**B.** Elastic Load Balancing.
**C.** Amazon EBS.
**D.** AWS Direct Connect.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Câu hỏi về dịch vụ tự động scale theo tăng trưởng web traffic.
- **Existing Resources:** Một web application.
- **Current Issue/Goal:** Cần dịch vụ xử lý lưu lượng tăng cao một cách tự động.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `automatically scale` | Dịch vụ phải có khả năng scale theo demand |
| `web traffic` | Lưu lượng HTTP/HTTPS đến ứng dụng web |
| `Elastic Load Balancing` | Tự động phân phối traffic đến nhiều instance; ALB/NLB scale theo traffic |
| `AWS CodePipeline` | CI/CD, không scale web traffic |
| `Amazon EBS` | Block storage, không scale traffic |
| `AWS Direct Connect` | Private network, không scale traffic |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Xác định dịch vụ scale
- **Constraints:** Web traffic increase

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: B**

**Giải thích:**
- `Elastic Load Balancing` (ELB: ALB, NLB, GLB) tự động phân phối incoming traffic across multiple targets (EC2, containers, IP addresses, Lambda functions) ở nhiều AZ.
- ELB **tự động scale** để đáp ứng lưu lượng tăng — không cần thao tác thủ công để thêm capacity cho load balancer.
- Kết hợp với `Auto Scaling group` để thêm/bớt EC2 backend theo demand.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- `AWS CodePipeline` là dịch vụ CI/CD để tự động hóa build/test/deploy, không xử lý web traffic runtime.
- Đáp án này phù hợp cho delivery pipeline.

**❌ Đáp án C:**
- `Amazon EBS` là block storage cho EC2. Khi instance scale up/down, EBS volume có thể được attach/detach nhưng EBS không tự scale theo web traffic.
- Đáp án này dùng cho storage layer.

**❌ Đáp án D:**
- `AWS Direct Connect` là private fiber từ on-prem DC đến AWS, băng thông cố định theo hợp đồng (1/10 Gbps). Không tự động scale theo web traffic.
- Đáp án này dùng cho hybrid connectivity.

## 6. MẺO GHI NHỚ (Memory Hook)
🧠 *Web traffic scale = `ELB` (phân phối) + `Auto Scaling` (thêm/bớt instance). `EBS` = storage, không phải load balancer.*
