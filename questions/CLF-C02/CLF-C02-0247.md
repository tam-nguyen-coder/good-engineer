# Câu hỏi #247

Which AWS Service offers an NFS file system that can be mounted concurrently from multiple EC2 instances?

## Options

**A.** Amazon Elastic File System.
**B.** Amazon Simple Storage Service.
**C.** Amazon Elastic Block Store.
**D.** AWS Storage Gateway.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Ngữ cảnh đề bài: _Which AWS Service offers an NFS file system that can be mounted concurrently from multiple EC2 instances?_
- **Existing Resources:** Đề nhắc tới: `EC2` — dịch vụ compute (máy chủ ảo) trên AWS Cloud.
- **Current Issue/Goal:** Hệ thống cần co giãn linh hoạt theo tải, không bị giới hạn bởi cấu hình cố định.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| ``EC2`` | dịch vụ compute (máy chủ ảo) trên AWS Cloud |
| ``S3`` | dịch vụ lưu trữ object bền vững (11x 9s) |
| ``EBS`` | Elastic Block Store - ổ đĩa block cho EC2 |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Scalability / Elasticity
- **Constraints:** Khả năng mở rộng theo tải

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: A**

**Giải thích:**
Đáp án `EFS` đúng vì Elastic File System - file system NFS managed, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `EFS` đóng vai trò: chia sẻ file giữa nhiều EC2, Linux, scale tự động.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án B:**
- Tại sao sai: `S3` (dịch vụ lưu trữ object bền vững (11x 9s)) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `S3`.

**❌ Đáp án C:**
- Tại sao sai: `EBS` (Elastic Block Store - ổ đĩa block cho EC2) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `EBS`.

**❌ Đáp án D:**
- Tại sao sai: `Storage Gateway` (cầu nối lưu trữ hybrid) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `Storage Gateway`.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **A** = `EC2`. Scale = Auto Scaling + Load Balancer, thêm/bớt instance theo tải.*
