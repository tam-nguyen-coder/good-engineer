# Câu hỏi #447

Which AWS services are defined as global instead of regional? (Select TWO)

## Options

**A.** Amazon Route 53.
**B.** Amazon EC2.
**C.** Amazon S3.
**D.** Amazon CloudFront.
**E.** Amazon DynamoDB.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Đề hỏi về khái niệm/dịch vụ AWS liên quan tới: _Which AWS services are defined as global instead of regional? (Select TWO)_
- **Existing Resources:** Không có tài nguyên cụ thể — câu hỏi thuộc dạng lý thuyết/khái niệm.
- **Current Issue/Goal:** Lựa chọn đáp án đúng nhất theo yêu cầu của đề.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `Global / Multi-region` | Phạm vi toàn cầu, nhiều region |
| ``EC2`` | dịch vụ compute (máy chủ ảo) trên AWS Cloud |
| ``S3`` | dịch vụ lưu trữ object bền vững (11x 9s) |
| ``DynamoDB`` | dịch vụ NoSQL key-value được quản lý hoàn toàn |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Networking / DNS / CDN
- **Constraints:** Tuân theo best practice AWS

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: A và D**

**Giải thích:**
Đề yêu cầu chọn nhiều đáp án. Hai phương án **A** và **D** cùng đáp ứng các tiêu chí của câu hỏi.
- **A** — `Route 53`: dịch vụ DNS và domain registration.
- **D** — `CloudFront`: CDN (Content Delivery Network) toàn cầu.

- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `S3`.
