# Câu hỏi #564

Which service provides a hybrid storage service that enables on-premises applications to seamlessly use cloud storage?

## Options

**A.** Amazon Glacier
**B.** AWS Snowball
**C.** AWS Storage Gateway
**D.** Amazon Elastic Block Storage (Amazon EBS)


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Ngữ cảnh đề bài: _Which service provides a hybrid storage service that enables on-premises applications to seamlessly use cloud storage?_
- **Existing Resources:** Không có tài nguyên cụ thể — câu hỏi thuộc dạng lý thuyết/khái niệm.
- **Current Issue/Goal:** Hệ thống cần co giãn linh hoạt theo tải, không bị giới hạn bởi cấu hình cố định.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `Hybrid / On-premises` | Kết nối môi trường on-premises với AWS |
| `Machine learning` | Ứng dụng ML/AI |

| ``EBS`` | Elastic Block Store - ổ đĩa block cho EC2 |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Scalability / Elasticity
- **Constraints:** Khả năng mở rộng theo tải

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: C**

**Giải thích:**
Đáp án `Storage Gateway` đúng vì cầu nối lưu trữ hybrid, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `Storage Gateway` đóng vai trò: kết nối on-premises với S3, Glacier qua cache local.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- Tại sao sai: `Glacier` (dịch vụ lưu trữ lạnh giá rẻ) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `Glacier`.

**❌ Đáp án B:**
- Tại sao sai: `Snowball/Snowmobile` (thiết bị vật lý chuyển dữ liệu quy mô lớn) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `Snowball/Snowmobile`.

**❌ Đáp án D:**
- Tại sao sai: `EBS` (Elastic Block Store - ổ đĩa block cho EC2) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `EBS`.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **C** = `Glacier`. Scale = Auto Scaling + Load Balancer, thêm/bớt instance theo tải.*
