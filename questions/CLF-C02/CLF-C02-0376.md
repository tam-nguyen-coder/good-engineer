# Câu hỏi #376

A company needs to host a big data application on AWS using EC2 instances. Which of the following AWS Storage services would they choose to automatically get high throughput to multiple compute nodes?

## Options

**A.** Amazon Elastic Block Store.
**B.** AWS Storage Gateway.
**C.** Amazon Elastic File System.
**D.** S3.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Tình huống được mô tả: _A company needs to host a big data application on AWS using EC2 instances. Which of the following AWS Storage services w..._
- **Existing Resources:** Đề nhắc tới: `EC2` — dịch vụ compute (máy chủ ảo) trên AWS Cloud.
- **Current Issue/Goal:** Lựa chọn giải pháp có hiệu năng cao nhất, độ trễ thấp nhất.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `Big data / Analytics` | Phân tích dữ liệu lớn |
| ``EC2`` | dịch vụ compute (máy chủ ảo) trên AWS Cloud |
| ``S3`` | dịch vụ lưu trữ object bền vững (11x 9s) |
| ``EBS`` | Elastic Block Store - ổ đĩa block cho EC2 |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Best performance / Lowest latency
- **Constraints:** Hiệu năng và độ trễ thấp

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: C**

**Giải thích:**
Đáp án `EFS` đúng vì Elastic File System - file system NFS managed, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `EFS` đóng vai trò: chia sẻ file giữa nhiều EC2, Linux, scale tự động.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- Tại sao sai: `EBS` (Elastic Block Store - ổ đĩa block cho EC2) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu lưu trữ persistent cho EC2, snapshot, multi-attach được ưu tiên hơn hiệu năng.

**❌ Đáp án B:**
- Tại sao sai: `Storage Gateway` (cầu nối lưu trữ hybrid) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu kết nối on-premises với S3, Glacier qua cache local được ưu tiên hơn hiệu năng.

**❌ Đáp án D:**
- Tại sao sai: `S3` (dịch vụ lưu trữ object bền vững (11x 9s)) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu lưu trữ file, backup, static website, data lake, log archive được ưu tiên hơn hiệu năng.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **C** = `EC2`. Hiệu năng = edge location + cache + chọn instance/region gần user.*
