# Câu hỏi #417

Which mechanism allows developers to access AWS sendees from application code?

## Options

**A.** AWS Software Development Kit.
**B.** AWS Management Console.
**C.** AWS CodePipeline.
**D.** AWS Config.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Câu hỏi về cơ chế giúp developer gọi AWS services từ code ứng dụng.
- **Existing Resources:** Có application code cần tương tác với AWS API.
- **Current Issue/Goal:** Tìm cơ chế/programmatic interface phù hợp.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `access AWS services from application code` | Cần programmatic access, không phải GUI |
| `AWS SDK` | Bộ thư viện cho nhiều ngôn ngữ (Python Boto3, JavaScript, Java, Go, .NET…) |
| `AWS Management Console` | GUI web, không dùng từ code được |
| `AWS CodePipeline` | CI/CD orchestration, không phải API access tool |
| `AWS Config` | Compliance service, không phải SDK |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Xác định cơ chế programmatic access
- **Constraints:** Phải dùng được từ application code

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: A**

**Giải thích:**
- `AWS SDK` (Software Development Kit) cung cấp API cho nhiều ngôn ngữ lập trình (Python `Boto3`, JavaScript v3, Java, Go, Ruby, .NET, PHP, C++), giúp application code gọi các AWS service một cách dễ dàng.
- SDK tự động xử lý: authentication, retries, pagination, signing requests, region endpoints…
- Đây là cách phổ biến nhất để embed AWS calls trong app, kèm `AWS CLI` cho môi trường scripting/shell.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án B:**
- `AWS Management Console` là giao diện web GUI, dành cho người dùng thao tác bằng chuột, không dùng được từ application code.
- Đáp án này phù hợp cho việc quản lý tài nguyên thủ công.

**❌ Đáp án C:**
- `AWS CodePipeline` là dịch vụ CI/CD để tự động hóa build/test/deploy, không phải cơ chế để code gọi AWS API.
- Đáp án này phù hợp cho delivery pipeline.

**❌ Đáp án D:**
- `AWS Config` theo dõi cấu hình tài nguyên và đánh giá compliance; không cung cấp API cho application code gọi AWS services.
- Đáp án này phù hợp cho configuration management.

## 6. MẺO GHI NHỚ (Memory Hook)
🧠 *Code gọi AWS = `SDK` (Boto3 cho Python). Script gọi AWS = `CLI`. Người dùng gọi AWS = `Management Console`.*
