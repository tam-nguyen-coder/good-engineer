# Câu hỏi #427

Which AWS services provide a way to extend an on-premises architecture to the AWS cloud? (Select TWO)

## Options

**A.** Amazon EBS.
**B.** Amazon Connect.
**C.** AWS Storage Gateway.
**D.** Amazon CloudFront.
**E.** AWS Direct Connect.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Câu hỏi multi-answer về các dịch vụ giúp mở rộng (extend) môi trường on-premises sang AWS Cloud.
- **Existing Resources:** Một on-premises architecture.
- **Current Issue/Goal:** Tìm dịch vụ hybrid để kết nối/extend on-prem lên AWS.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `extend on-premises architecture` | Hybrid bridge giữa DC và AWS |
| `AWS Storage Gateway` | Hybrid storage: volume, tape, file gateway — kết nối on-prem storage với S3/Glacier/EBS |
| `AWS Direct Connect` | Private fiber line từ on-prem DC đến AWS |
| `Amazon EBS` | Block storage cho EC2 trong AWS, không phải hybrid |
| `Amazon Connect` | Cloud contact center, không phải hybrid bridge |
| `Amazon CloudFront` | CDN, không phải hybrid bridge on-prem |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Xác định dịch vụ hybrid (multi-answer)
- **Constraints:** Phải giúp mở rộng on-prem sang AWS

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: C và E**

**Giải thích:**
- `AWS Storage Gateway`: hybrid storage appliance chạy trong on-prem DC, kết nối với `Amazon S3`/`Glacier`/`EBS` qua kênh mã hóa. Cho phép on-prem app dùng cloud storage mà không cần thay đổi code. Có 3 loại: File Gateway, Volume Gateway, Tape Gateway.
- `AWS Direct Connect`: thiết lập kết nối vật lý riêng (private fiber) giữa on-prem DC và AWS Region, mang lại băng thông ổn định, latency thấp, chi phí data transfer thấp hơn internet.
- Cả hai đều là giải pháp hybrid phổ biến giúp extend on-prem lên cloud.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- `Amazon EBS` là block storage gắn vào EC2 instance trong AWS. Không có vai trò kết nối on-prem.
- Đáp án này thuộc AWS-only storage.

**❌ Đáp án B:**
- `Amazon Connect` là cloud contact center (call center), không phải dịch vụ kết nối hybrid.
- Đáp án này dùng cho customer service/voice solution.

**❌ Đáp án D:**
- `Amazon CloudFront` là CDN phân phối nội dung tĩnh/dynamic đến edge locations. Không phải hybrid bridge cho on-prem.
- Đáp án này dùng cho caching/acceleration nội dung.

## 6. MẺO GHI NHỚ (Memory Hook)
🧠 *Extend on-prem sang AWS = `Storage Gateway` (storage hybrid) + `Direct Connect` (private network). `Site-to-Site VPN` cũng là hybrid network qua internet.*
