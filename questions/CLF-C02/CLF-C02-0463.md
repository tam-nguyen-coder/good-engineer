# Câu hỏi #463

Which AWS services should be used for read/write of constantly changing data? (Select TWO)

## Options

**A.** Amazon Glacier.
**B.** Amazon RDS.
**C.** AWS Snowball.
**D.** Amazon Redshift.
**E.** Amazon EFS.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Đề hỏi về khái niệm/dịch vụ AWS liên quan tới: _Which AWS services should be used for read/write of constantly changing data? (S..._
- **Existing Resources:** Không có tài nguyên cụ thể — câu hỏi thuộc dạng lý thuyết/khái niệm.
- **Current Issue/Goal:** Lựa chọn đáp án đúng nhất theo yêu cầu của đề.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| ``RDS`` | dịch vụ database quan hệ được quản lý |
| ``Glacier`` | dịch vụ lưu trữ lạnh giá rẻ |
| ``EFS`` | Elastic File System - file system NFS managed |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Database
- **Constraints:** Tuân theo best practice AWS

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: B và E**

**Giải thích:**
Đề yêu cầu chọn nhiều đáp án. Hai phương án **B** và **E** cùng đáp ứng các tiêu chí của câu hỏi.
- **B** — `RDS`: dịch vụ database quan hệ được quản lý.
- **E** — `EFS`: Elastic File System - file system NFS managed.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- Tại sao sai: `Glacier` (dịch vụ lưu trữ lạnh giá rẻ) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `Glacier`.

**❌ Đáp án C:**
- Tại sao sai: `Snowball/Snowmobile` (thiết bị vật lý chuyển dữ liệu quy mô lớn) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `Snowball/Snowmobile`.

**❌ Đáp án D:**
- Tại sao sai: `Redshift` (data warehouse columnar) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng trong tình huống khác yêu cầu cụ thể `Redshift`.
