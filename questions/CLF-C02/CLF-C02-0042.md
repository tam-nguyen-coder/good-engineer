# Câu hỏi #42

What is the AWS service that enables AWS architects to manage infrastructure as code?

## Options

**A.** AWS CloudFormation.
**B.** AWS Config.
**C.** Amazon SES.
**D.** Amazon EMR.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Đề hỏi về dịch vụ cho phép AWS architect quản lý hạ tầng dưới dạng **code** (Infrastructure as Code — IaC).
- **Existing Resources:** Hạ tầng cần được định nghĩa và provision tự động.
- **Current Issue/Goal:** Chọn service chuẩn cho IaC trên AWS.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `infrastructure as code (IaC)` | Định nghĩa, provision và quản lý hạ tầng qua file code/template |
| `AWS CloudFormation` | Service IaC chính thống của AWS, dùng JSON/YAML template |
| `AWS Config` | Theo dõi config và compliance, không phải IaC |
| `Amazon SES` | Gửi email, không liên quan IaC |
| `Amazon EMR` | Big data processing, không liên quan IaC |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Khái niệm — IaC service
- **Constraints:** Phải là service cho phép quản lý infra bằng code

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: A**

**Giải thích:**
- `AWS CloudFormation` là dịch vụ cho phép mô tả và provision tất cả AWS resource trong một **template** (JSON hoặc YAML), từ đó tạo ra **stack** một cách nhất quán và lặp lại được.
- Hỗ trợ hầu hết AWS resource: EC2, VPC, RDS, S3, IAM, Lambda, ... và nhiều tính năng nâng cao: `StackSets` (multi-account/multi-region), `Nested Stacks`, `Drift Detection`, `Change Sets`.
- Công cụ IaC chuẩn mực, đi kèm AWS CDK (Cloud Development Kit) cho phép viết IaC bằng ngôn ngữ lập trình (TypeScript, Python, Java, Go, ...).

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án B:**
- `AWS Config` theo dõi và đánh giá cấu hình resource so với baseline, giúp audit compliance. Không có chức năng provision resource từ template.

**❌ Đáp án C:**
- `Amazon SES` (Simple Email Service) cung cấp dịch vụ gửi email transactional/marketing, không liên quan IaC.

**❌ Đáp án D:**
- `Amazon EMR` (Elastic MapReduce) là dịch vụ big data với Hadoop/Spark, không phải công cụ quản lý infrastructure.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *"Infrastructure as Code trên AWS = `CloudFormation` (template JSON/YAML) hoặc `AWS CDK`. `Config` = audit, `SES` = email, `EMR` = big data."*
