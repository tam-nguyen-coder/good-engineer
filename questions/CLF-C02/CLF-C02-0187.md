# Câu hỏi #187

Which of the following activities supports the Operational Excellence pillar of the AWS Well-Architected Framework?

## Options

**A.** Using AWS Trusted Advisor to find underutilized resources.
**B.** Using AWS CloudTrail to record user activities.
**C.** Using AWS CloudFormation to manage infrastructure as code.
**D.** Deploying an application in multiple Availability Zones.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Ngữ cảnh đề bài: _Which of the following activities supports the Operational Excellence pillar of the AWS Well-Architected Framework?_
- **Existing Resources:** Đề nhắc tới: `Well-Architected Tool` — đánh giá kiến trúc theo 6 trụ cột.
- **Current Issue/Goal:** Cần giám sát, theo dõi và ghi nhận hoạt động của hệ thống/tài khoản.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| ``CloudTrail`` | dịch vụ audit API call |
| ``CloudFormation`` | Infrastructure as Code |
| ``Trusted Advisor`` | cố vấn tối ưu chi phí, bảo mật, hiệu năng |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Monitoring / Auditing
- **Constraints:** Tuân theo best practice AWS

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: C**

**Giải thích:**
Đáp án `CloudFormation` đúng vì Infrastructure as Code, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `CloudFormation` đóng vai trò: định nghĩa hạ tầng bằng template JSON/YAML, tự động provision.

- Tại sao sai: Đáp án này không trực tiếp giải quyết yêu cầu cụ thể của đề, hoặc dịch vụ/tính năng này có giới hạn so với đáp án đúng.
