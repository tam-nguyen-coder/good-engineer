# Câu hỏi #23

Which of the following does NOT belong to the AWS Cloud Computing models?

## Options

**A.** Platform as a Service (PaaS).
**B.** Infrastructure as a Service (IaaS).
**C.** Software as a Service (SaaS).
**D.** Networking as a Service (NaaS).


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Đề hỏi về các mô hình cloud computing. Cần chọn mô hình **KHÔNG** thuộc nhóm chính thống.
- **Existing Resources:** Không có — câu hỏi lý thuyết.
- **Current Issue/Goal:** Nhận diện 3 mô hình cloud computing chuẩn.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `IaaS` | Cung cấp hạ tầng (EC2, VPC, EBS) — khách hàng lo OS, runtime, app |
| `PaaS` | Cung cấp platform (RDS, Beanstalk, Lambda) — AWS lo OS, runtime |
| `SaaS` | Phần mềm hoàn chỉnh (WorkMail, Chime, QuickSight) |
| `NaaS` | Không phải mô hình cloud computing chính thống |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Khái niệm — Cloud computing models
- **Constraints:** Chọn mô hình KHÔNG thuộc nhóm chính

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: D**

**Giải thích:**
- Ba mô hình cloud computing chính thống được công nhận rộng rãi là `IaaS`, `PaaS`, và `SaaS`.
- `Networking as a Service (NaaS)` không phải là mô hình cloud computing chính thức. Network trong cloud thường được cung cấp như một phần của IaaS (VPC, Direct Connect, VPN) chứ không tách thành một mô hình riêng.
- Một số tài liệu đề cập NaaS theo nghĩa "network infrastructure delivered as a service" (SD-WAN, CDN) nhưng không nằm trong ba mô hình kinh điển.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- `PaaS` (Platform as a Service) là mô hình cloud chính thống: AWS cung cấp platform, khách hàng chỉ lo app và data. Ví dụ: RDS, Elastic Beanstalk, Lambda.

**❌ Đáp án B:**
- `IaaS` (Infrastructure as a Service) là mô hình cloud chính thống: cung cấp hạ tầng ảo hóa (EC2, VPC, EBS). Ví dụ: AWS EC2.

**❌ Đáp án C:**
- `SaaS` (Software as a Service) là mô hình cloud chính thống: phần mềm hoàn chỉnh chạy trên cloud. Ví dụ: AWS Chime, WorkMail, Amazon Connect.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *"Cloud computing có 3 mô hình: `IaaS`, `PaaS`, `SaaS`. `NaaS` không nằm trong nhóm chính — đó là bẫy của câu hỏi."*
