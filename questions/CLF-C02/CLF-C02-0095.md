# Câu hỏi #95

Which AWS services can be used to improve the performance of a global application and reduce latency for its users? (Choose TWO)

## Options

**A.** AWS KMS.
**B.** AWS Global accelerator.
**C.** AWS Direct Connect.
**D.** AWS Glue.
**E.** Amazon CloudFront.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Ngữ cảnh đề bài: _Which AWS services can be used to improve the performance of a global application and reduce latency for its users? (Cho..._
- **Existing Resources:** Không có tài nguyên cụ thể — câu hỏi thuộc dạng lý thuyết/khái niệm.
- **Current Issue/Goal:** Lựa chọn giải pháp có hiệu năng cao nhất, độ trễ thấp nhất.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `Global / Multi-region` | Phạm vi toàn cầu, nhiều region |
| ``CloudFront`` | CDN (Content Delivery Network) toàn cầu |
| ``KMS`` | Key Management Service |
| ``Direct Connect`` | kết nối mạng riêng tốc độ cao tới AWS |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Best performance / Lowest latency
- **Constraints:** Hiệu năng và độ trễ thấp

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: B và E**

**Giải thích:**
Đề yêu cầu chọn nhiều đáp án. Hai phương án **B** và **E** cùng đáp ứng các tiêu chí của câu hỏi.
- **B** — AWS Global accelerator..
- **E** — `CloudFront`: CDN (Content Delivery Network) toàn cầu.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- Tại sao sai: `KMS` (Key Management Service) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu quản lý encryption key để mã hóa dữ liệu được ưu tiên hơn hiệu năng.

**❌ Đáp án C:**
- Tại sao sai: `Direct Connect` (kết nối mạng riêng tốc độ cao tới AWS) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu đường truyền dedicated từ on-premises tới AWS, giảm chi phí bandwidth được ưu tiên hơn hiệu năng.

**❌ Đáp án D:**
- Tại sao sai: `Glue` (ETL service managed) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu chuẩn bị, biến đổi, nạp dữ liệu cho analytics được ưu tiên hơn hiệu năng.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **B và E** = `CloudFront`. Hiệu năng = edge location + cache + chọn instance/region gần user.*
