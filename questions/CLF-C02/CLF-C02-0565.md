# Câu hỏi #565

Where should a company go to search software listings from independent software vendors to find, test, buy and deploy software that runs on AWS?

## Options

**A.** AWS Marketplace.
**B.** Amazon Lumberyard.
**C.** AWS Artifact.
**D.** Amazon CloudSearch.

Trong ngữ cảnh này, `Marketplace` đóng vai trò: tìm, mua, triển khai phần mềm của đối tác AWS.
