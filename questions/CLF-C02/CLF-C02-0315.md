# Câu hỏi #315

An organization needs to build a financial application that requires support for ACID transactions. Which AWS database service is most appropriate in this case?

## Options

**A.** RedShift.
**B.** RDS.
**C.** CloudHSM.
**D.** DMS.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Tình huống được mô tả: _An organization needs to build a financial application that requires support for ACID transactions. Which AWS database s..._
- **Existing Resources:** Không có tài nguyên cụ thể — câu hỏi thuộc dạng lý thuyết/khái niệm.
- **Current Issue/Goal:** Lựa chọn đáp án đúng nhất theo yêu cầu của đề.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| ``RDS`` | dịch vụ database quan hệ được quản lý |
| ``Redshift`` | data warehouse columnar |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Database
- **Constraints:** Tuân theo best practice AWS

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: B**

**Giải thích:**
Đáp án `RDS` đúng vì dịch vụ database quan hệ được quản lý, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `RDS` đóng vai trò: chạy SQL database (MySQL, PostgreSQL, Oracle, SQL Server, MariaDB, Aurora).
