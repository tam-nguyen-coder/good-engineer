# Câu hỏi #226

An external auditor is requesting a log of all accesses to the AWS resources in the company's account. Which of the following services will provide the auditor with the requested information?

## Options

**A.** AWS CloudTrail.
**B.** Amazon CloudFront.
**C.** AWS CloudFormation.
**D.** Amazon CloudWatch.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Tình huống được mô tả: _An external auditor is requesting a log of all accesses to the AWS resources in the company's account. Which of the foll..._
- **Existing Resources:** Không có tài nguyên cụ thể — câu hỏi thuộc dạng lý thuyết/khái niệm.
- **Current Issue/Goal:** Lựa chọn giải pháp bảo mật tốt nhất theo best practice của AWS.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `Audit / Compliance` | Theo dõi, truy vết, kiểm toán |
| ``CloudWatch`` | dịch vụ monitoring và observability |
| ``CloudTrail`` | dịch vụ audit API call |
| ``CloudFront`` | CDN (Content Delivery Network) toàn cầu |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Most secure / Security best practice
- **Constraints:** Bảo mật dữ liệu và quyền truy cập

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: A**

**Giải thích:**
Đáp án `CloudTrail` đúng vì dịch vụ audit API call, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `CloudTrail` đóng vai trò: ghi lại mọi API call vào tài khoản AWS, hỗ trợ kiểm tra bảo mật.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án B:**
- Tại sao sai: `CloudFront` (CDN (Content Delivery Network) toàn cầu) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu bảo mật chính là phân phối nội dung tĩnh/động qua edge location, giảm latency này.

**❌ Đáp án C:**
- Tại sao sai: `CloudFormation` (Infrastructure as Code) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu bảo mật chính là định nghĩa hạ tầng bằng template JSON/YAML, tự động provision này.

**❌ Đáp án D:**
- Tại sao sai: `CloudWatch` (dịch vụ monitoring và observability) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu bảo mật chính là thu thập metric, log, set alarm, dashboard này.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **A** = `CloudWatch`. Bảo mật = defense-in-depth: nhiều lớp, nguyên tắc least privilege.*
