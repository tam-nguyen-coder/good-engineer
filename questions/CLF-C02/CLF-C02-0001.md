# Câu hỏi #1

AWS allows users to manage their resources using a web based user interface. What is the name of this interface?

## Options

**A.** AWS CLI.
**B.** AWS API.
**C.** AWS SDK.
**D.** AWS Management Console.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Đề hỏi về giao diện quản lý tài nguyên AWS dựa trên nền tảng web.
- **Existing Resources:** Không có tài nguyên cụ thể — đây là câu hỏi lý thuyết về các công cụ tương tác với AWS.
- **Current Issue/Goal:** Xác định đúng tên gọi của giao diện web do AWS cung cấp để người dùng quản lý tài nguyên.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `web based user interface` | Giao diện đồ họa trên trình duyệt, không phải dòng lệnh hay API |
| `AWS CLI` | Command Line Interface — giao diện dòng lệnh |
| `AWS SDK` | Bộ thư viện lập trình để tích hợp AWS vào code |
| `AWS Management Console` | Giao diện web (GUI) chính thức của AWS |
| `AWS API` | Giao diện lập trình ứng dụng để gọi trực tiếp các dịch vụ |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Khái niệm cơ bản (Definition / terminology)
- **Constraints:** Phải là giao diện web (GUI), không phải CLI, SDK hay API

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: D**

**Giải thích:**
- `AWS Management Console` chính là giao diện web mà AWS cung cấp tại địa chỉ `https://console.aws.amazon.com`, cho phép người dùng quản lý hầu hết các dịch vụ AWS thông qua trình duyệt mà không cần viết code.
- Đây là công cụ thân thiện nhất với người dùng mới, hỗ trợ thao tác trỏ-clich cho hầu hết resource (EC2, S3, RDS, v.v.).
- Các lựa chọn khác là CLI (dòng lệnh), SDK (thư viện lập trình), API (giao diện lập trình) — đều không phải giao diện web.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- `AWS CLI` là công cụ dòng lệnh, người dùng phải gõ lệnh trong terminal để quản lý tài nguyên, không phải giao diện web.

**❌ Đáp án B:**
- `AWS API` là tập các endpoint HTTP/HTTPS mà AWS cung cấp để gọi dịch vụ theo phương thức lập trình, không phải giao diện đồ họa.

**❌ Đáp án C:**
- `AWS SDK` (Software Development Kit) là bộ thư viện cho nhiều ngôn ngữ (Python boto3, JavaScript, Java, ...) dùng để viết code tương tác AWS, không phải giao diện web.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *"Web GUI = `Management Console`; CLI = dòng lệnh; SDK/API = lập trình. Thấy 'web based' thì chọn `Management Console`."*
