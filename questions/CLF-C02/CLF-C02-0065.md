# Câu hỏi #65

What is the AWS service that provides a virtual network dedicated to your AWS account?

## Options

**A.** AWS VPN.
**B.** AWS Subnets.
**C.** AWS Dedicated Hosts.
**D.** Amazon VPC.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Đề hỏi về khái niệm/dịch vụ AWS liên quan tới: _What is the AWS service that provides a virtual network dedicated to your AWS ac..._
- **Existing Resources:** Không có tài nguyên cụ thể — câu hỏi thuộc dạng lý thuyết/khái niệm.
- **Current Issue/Goal:** Xác định đúng khái niệm/dịch vụ/tính năng AWS được hỏi tới.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| ``VPC`` | Virtual Private Cloud - mạng riêng ảo trong AWS |
| ``Site-to-Site VPN`` | kết nối mã hóa qua internet tới AWS |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Networking / DNS / CDN
- **Constraints:** Tuân theo best practice AWS

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: D**

**Giải thích:**
Đáp án `VPC` đúng vì Virtual Private Cloud - mạng riêng ảo trong AWS, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `VPC` đóng vai trò: cô lập tài nguyên mạng, kiểm soát subnet, routing, security group.
