# Câu hỏi #35

How can you view the distribution of AWS spending in one of your AWS accounts?

## Options

**A.** By using Amazon VPC console.
**B.** By contacting the AWS Support team.
**C.** By using AWS Cost Explorer.
**D.** By contacting the AWS Finance team.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Cần xem **phân bổ chi tiêu AWS** trong account.
- **Existing Resources:** AWS account có sử dụng nhiều dịch vụ.
- **Current Issue/Goal:** Có visualization về AWS spending distribution.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `distribution of AWS spending` | Phân bổ chi phí theo service, region, account, ... |
| `AWS Cost Explorer` | Công cụ visualize cost & usage, có chart và filter |
| `VPC console` | Quản lý network VPC, không liên quan billing |
| `AWS Support` | Hỗ trợ kỹ thuật, không phải dashboard visualization |
| `AWS Finance` | Không phải kênh chính thức để user xem cost |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Chọn công cụ xem cost visualization
- **Constraints:** Phải là self-service dashboard

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: C**

**Giải thích:**
- `AWS Cost Explorer` là công cụ cho phép visualize, phân tích và dự đoán chi phí AWS một cách trực quan.
- Hỗ trợ filter theo `service`, `region`, `linked account`, `tag`, `instance type`, ... và group theo nhiều tiêu chí để xem phân bổ spending.
- Có chart bar/line/stacked area, có thể drill-down từng khoảng thời gian (daily, monthly, ...).
- Lưu trữ 12 tháng dữ liệu lịch sử và dự đoán 12 tháng tương lai.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án A:**
- `Amazon VPC console` dùng để quản lý VPC, subnet, route table, security group, ... — không liên quan đến billing.

**❌ Đáp án B:**
- `AWS Support team` chuyên hỗ trợ kỹ thuật, không phải công cụ visualization chi phí.

**❌ Đáp án D:**
- Không có kênh "AWS Finance" chính thức để user thường xem cost. Cost Explorer là self-service, miễn phí, truy cập trực tiếp trong console.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *"Xem phân bổ cost = `Cost Explorer` (chart + filter). `Budgets` = alert, `CUR` = data chi tiết, `VPC console` = network. Đừng nhầm!"*
