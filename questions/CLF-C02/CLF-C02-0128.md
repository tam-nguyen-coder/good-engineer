# Câu hỏi #128

What is the AWS data warehouse service that supports a high level of query performance on large amounts of datasets?

## Options

**A.** Amazon Redshift.
**B.** Amazon Kinesis.
**C.** Amazon DynamoDB.
**D.** Amazon RDS.


---

## 1. CONTEXT & ĐỀ BÀI
- **Scenario:** Ngữ cảnh đề bài: _What is the AWS data warehouse service that supports a high level of query performance on large amounts of datasets?_
- **Existing Resources:** Đề nhắc tới: `Redshift` — data warehouse columnar.
- **Current Issue/Goal:** Lựa chọn giải pháp có hiệu năng cao nhất, độ trễ thấp nhất.

## 2. KEYWORDS QUAN TRỌNG
| Keyword | Ý nghĩa / Gợi ý |
|---------|-----------------|
| `Big data / Analytics` | Phân tích dữ liệu lớn |
| ``RDS`` | dịch vụ database quan hệ được quản lý |
| ``DynamoDB`` | dịch vụ NoSQL key-value được quản lý hoàn toàn |
| ``Kinesis`` | streaming data service |

## 3. YÊU CẦU CỦA ĐỀ
- **Question type:** Best performance / Lowest latency
- **Constraints:** Hiệu năng và độ trễ thấp

## 4. ĐÁP ÁN ĐÚNG
**✅ Đáp án: A**

**Giải thích:**
Đáp án `Redshift` đúng vì data warehouse columnar, phù hợp với yêu cầu của đề.
Trong ngữ cảnh này, `Redshift` đóng vai trò: phân tích dữ liệu lớn, OLAP, BI.

## 5. CÁC ĐÁP ÁN SAI
**❌ Đáp án B:**
- Tại sao sai: `Kinesis` (streaming data service) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu thu thập, xử lý, phân tích dữ liệu streaming thời gian thực được ưu tiên hơn hiệu năng.

**❌ Đáp án C:**
- Tại sao sai: `DynamoDB` (dịch vụ NoSQL key-value được quản lý hoàn toàn) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu lưu trữ dữ liệu phi quan hệ với độ trễ thấp ở mọi quy mô được ưu tiên hơn hiệu năng.

**❌ Đáp án D:**
- Tại sao sai: `RDS` (dịch vụ database quan hệ được quản lý) không phù hợp với yêu cầu cụ thể của đề.
- Khi nào đúng: Đáp án này sẽ đúng khi yêu cầu chạy SQL database (MySQL, PostgreSQL, Oracle, SQL Server, MariaDB, Aurora) được ưu tiên hơn hiệu năng.

## 6. MẸO GHI NHỚ (Memory Hook)
🧠 *Đáp án **A** = `RDS`. Hiệu năng = edge location + cache + chọn instance/region gần user.*
