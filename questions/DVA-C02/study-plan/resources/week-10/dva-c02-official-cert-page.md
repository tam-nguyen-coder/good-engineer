## 🎯 Điểm thi quan trọng (tóm tắt tiếng Việt)
- **Số liệu kỳ thi phải thuộc:** **65 câu** · **130 phút** · **$150 USD** · điểm đậu **720/1000**. Đây chính là các con số bạn canh giờ khi chạy mock (~2 phút/câu).
- **Định dạng câu:** multiple choice (1 đáp án đúng) hoặc multiple response (chọn 2–3 trong ≥5). Có **câu không tính điểm** (unscored) để AWS thử nghiệm — đừng hoảng khi gặp câu lạ.
- **Hình thức thi:** **Pearson VUE test center** HOẶC **online proctored** (thi tại nhà, có giám thị qua webcam). Chọn hình thức trước khi đặt lịch — xem file chính sách để biết yêu cầu phòng/thiết bị.
- **Ngôn ngữ:** English, Japanese, Korean, Portuguese (Brazil), Simplified Chinese, Spanish (Latin America). Người không phải bản ngữ tiếng Anh có thể xin **ESL +30 phút**.
- **Kinh nghiệm khuyến nghị:** ~1 năm hands-on phát triển/vận hành ứng dụng trên AWS. Chưa có nền IT thì nên lấy Cloud Practitioner trước.
- **Lộ trình ôn 4 bước của AWS:** (1) đọc exam guide + làm **Official Practice Question Set**; (2) học digital courses + Builder Labs/Cloud Quest/Jam; (3) luyện câu hỏi + flashcard theo domain; (4) làm **Official Practice Exam** để đo mức sẵn sàng.
- **Hiệu lực & ưu đãi:** chứng chỉ có giá trị **3 năm**; recert bằng cách thi lại bản mới nhất hoặc đạt DevOps Engineer – Professional. Đậu 1 chứng chỉ → **giảm 50%** phí thi kế tiếp.

## 📄 Nội dung (trích từ tài liệu gốc)

# AWS Certified Developer - Associate

## Overview

The AWS Certified Developer - Associate certification validates technical proficiency in developing, testing, deploying, and debugging AWS Cloud-based applications. This associate-level credential demonstrates skills in application development, optimization, packaging, deployment, CI/CD workflows, and application issue resolution.

## Exam Details

| Aspect | Information |
|--------|-------------|
| **Category** | Associate |
| **Duration** | 130 minutes |
| **Format** | 65 questions (multiple choice or multiple response) |
| **Cost** | $150 USD |
| **Testing Options** | Pearson VUE testing center or online proctored exam |
| **Languages** | English, Japanese, Korean, Portuguese (Brazil), Simplified Chinese, Spanish (Latin America) |

> Ghi chú: điểm đậu (passing score) là **720/1000** — xem chi tiết trong file `dva-c02-exam-guide.md`.

## Target Audience

Per the exam guide, candidates should have "one or more years of prior hands-on experience" developing and maintaining AWS applications. This certification suits individuals in IT or cloud developer roles seeking an entry point to the AWS Certification journey.

Candidates without IT work experience may benefit from first earning the AWS Certified Cloud Practitioner credential before attempting this exam.

## Exam Preparation

The four-step preparation plan includes:

1. **Familiarize with exam structure** – Review the official exam guide and take the AWS Certification Official Practice Question Set
2. **Enhance knowledge** – Enroll in digital courses, practice with AWS Builder Labs, AWS Cloud Quest, and AWS Jam
3. **Review and practice** – Explore exam domains, reinforce knowledge with exam-style questions and flashcards
4. **Assess readiness** – Complete the AWS Certification Official Practice Exam

## Certification Validity & Advancement

This certification remains valid for three years. Recertification occurs by retaking the latest exam version or earning the AWS Certified DevOps Engineer - Professional credential.

Upon earning one AWS Certification, candidates receive 50% discount on subsequent AWS Certification exams.

Advancement paths include the AWS Certified SysOps Administrator - Associate, AWS Certified DevOps Engineer - Professional, or AWS Certified Data Engineer - Associate.
