CORRECT ANSWER

AWS Management Console

Web-Based GUI: Official portal to manage AWS services through standard browsers.
User Friendly: Visual point-and-click interface, great for beginners.
Zero Coding: Access resources directly without writing code or terminal commands.
A
INCORRECT

AWS CLI

Command Line: A command-line tool for executing statements.
Terminal Only: Text-based interaction, not a web interface.
B
INCORRECT

AWS API

Backend Interface: Programming endpoints used for backend communications.
HTTP Requests: Used programmatically, not via a browser UI.
C
INCORRECT

AWS SDK

Software Kits: Development libraries for specific coding languages.
Integration: Used by programmers to build custom applications.
