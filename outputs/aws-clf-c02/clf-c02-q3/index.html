CORRECT ANSWER

AWS CloudTrail

Activity Logging: Tracks user activity, API calls, and resource mutations within your AWS account.
Audit Trails: Log files reveal who called the API (e.g. TerminateInstances), when, and from what IP.
A
INCORRECT

Amazon Inspector

Vulnerability Scans: Scans operating systems and EC2 packages for known security flaws, not user actions.
C
INCORRECT

AWS Trusted Advisor

Optimization Advisor: Advises on security, cost, performance, and limits, but does not provide event logs.
D
INCORRECT

EC2 Instance Usage Report

Cost & Usage Metrics: Aggregates historical runtime hours and billing calculations, not audit tracking.
